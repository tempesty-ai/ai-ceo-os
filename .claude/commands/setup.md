# AI CEO OS 셋업 스킬

> 이 스킬은 사용자와 대화하며 사업 정보를 수집하고, 맞춤형 AI 경영 시스템을 자동으로 구축합니다.
> 상세 설계: `skills/setup/SKILL.md` 참조

## 실행 흐름

```
Step 1: 환영 + 기존 파일 체크
  ↓
Step 2: 사업 인터뷰 (5~7개 질문, 한 번에 하나씩)
  ↓
Step 3: 파일 자동 생성 (7개)
  ↓
Step 4: AI 비서 프로필 카드 출력
  ↓
Step 5: 첫 번째 미션 즉시 실행
```

## 실행

`skills/setup/SKILL.md` 파일을 읽고 그 안의 지침을 **정확히** 따라 실행하십시오.

핵심 규칙:
1. **한 질문씩만** 던질 것 (한번에 여러 질문 금지)
2. **중립 코치 톤** (특정 브랜드 톤 아님)
3. 모든 질문에 **업종 예시 3~5개** 포함
4. **"모르겠어요" = 유효한 답** → 기본값 자동 적용
5. 파일 생성 후 **프로필 카드** 출력 → **첫 미션 즉시 제안**
