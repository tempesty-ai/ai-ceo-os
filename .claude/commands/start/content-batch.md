# /start:content-batch — 화요일 콘텐츠 배치 생성

> 매주 화요일 실행. 이번 주 확정된 주제 3개 기반으로 대본 + 캡션 한 번에 생성.

## 실행 전 확인
- `tasks/todo.md`에서 `/start:weekly-start` 결과 확인 (주제 3개 + CTA 방향)
- 주제별 콘텐츠 축 확인: AI+인사이트 50% / AI+육아 50%

## 실행 순서

### Phase 1: AI+인사이트 릴스 대본 (주 3회 기준 1~2개)
각 대본마다:
1. 후킹 유형 선택 (숫자충격/금지경고/타겟지목/아무도안알려줬다형)
2. 대본 구조 선택 (A:문제→해결책 / B:카운트다운 / C:반전서사)
3. 15~30초 분량으로 작성
4. 자막 텍스트 별도 표기
5. 캡션 작성 (후킹 1줄 + 가치 2~3줄 + 키워드 + CTA 1개 + 해시태그 3~5개)

### Phase 2: AI+육아 릴스 대본 (주 3회 기준 1~2개)
1. 타겟 감정: "나도 이렇게 하면 되겠다" 공감형
2. 후킹: "육아하는 부모님" 타겟 지목형 우선
3. 대본 구조 선택 후 작성
4. 캡션 작성

### Phase 3: 검수
- CTA가 대본마다 1개인지 확인
- 첫 3초 후킹 작동 여부 확인
- 3개 대본의 CTA 유형이 다양한지 확인 (저장/댓글/팔로우 분산)
