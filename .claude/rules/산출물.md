# 산출물 저장 규칙

> outputs/ 폴더 작업 시 자동 적용됩니다.

## 폴더 구조

**경로**: `outputs/{연도}/{월}/{일}/[유형]_제목.md`

예시:
```
outputs/2026/04/07/[릴스]_AI활용법_30초_대본.md
outputs/2026/04/07/[분석]_3월_매출_보고서.md
```

## 유형 태그 (파일명 접두사)

| 태그 | 용도 | 예시 |
|:-----|:-----|:-----|
| `[릴스]` | 릴스/숏폼 대본 | `[릴스]_교육형_대본.md` |
| `[유튜브]` | 유튜브 기획/원고 | `[유튜브]_제미나이_기능_원고.md` |
| `[전략]` | 전략서/비전 문서 | `[전략]_3월_매출_전략.md` |
| `[기획]` | 기획서/제안서 | `[기획]_챌린지_커리큘럼.md` |
| `[분석]` | 데이터 분석 보고서 | `[분석]_2월_성과.md` |
| `[강의]` | 강의 설계/커리큘럼 | `[강의]_AI입문.md` |
| `[세일즈]` | 세일즈 자료 | `[세일즈]_B2B_제안서.md` |
| `[회고]` | 회고/일지 | `[회고]_2월_프로젝트.md` |
| `[기타]` | 위에 해당 없음 | `[기타]_캐러셀_원고.md` |

## 왜 이 구조인가

1. **시간순 자동 정렬**: 오래된 산출물 찾기 쉬움
2. **태그로 분류**: "릴스만 보고 싶다" 즉시 grep 가능
3. **AI가 자동 저장**: 김이사가 결과물 만들면 이 경로로 자동 저장
4. **백업 안전**: 날짜별 폴더는 백업 시스템과 잘 맞음
