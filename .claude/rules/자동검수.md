# 자동검수 규칙

> 모든 스킬 실행 후 자동 적용. CEO가 별도 명령어 칠 필요 없음.

## 동작 방식

**스킬 실행 → 산출물 납품 → 검수 기준 자동 체크 → 결과 1줄 첨부**

### 전항목 합격 시
산출물 마지막에 1줄:
```
📋 검수: 전항목 합격
```

### 불합격 항목이 있을 때
산출물 마지막에 플래그:
```
📋 검수: [N]건 플래그
- ❌ [기준ID] [기준명]: [구체적 이유]
→ 수정할까요?
```

## 적용 범위
- 모든 스킬 — 예외 없음
- 스킬의 `## 검수 기준` 섹션에 정의된 기준만 체크
- 검수 기준이 없는 스킬은 체크 생략

## 체크 규칙
1. **필수 항목**: 있다/없다 이진 체크만 (주관적 품질 판단 금지)
2. **권장 항목**: 불합격이어도 플래그하지 않음
3. **자기 채점 편향 방지**: "잘했다"는 절대 표시하지 않음. 문제만 플래그.

## 스킬 작성 시
새 스킬을 만들 때 `## 검수 기준` 섹션을 추가하면 자동으로 체크됩니다:

```markdown
## 검수 기준
| ID | 기준 | 합격 조건 | 가중치 |
|:---|:---|:---|:---:|
| Q1 | 첫 문장 후킹 | 3초 내 시선 잡는 문장 | 필수 |
| Q2 | 길이 | 30~40초 분량 | 필수 |
```

## CEO 피드백 → 학습 기록
CEO가 결과물에 피드백을 주면 해당 스킬의 `## 학습 기록` 섹션에 자동 추가됩니다.
다음 실행부터 반영됩니다.
