# 산출물 품질 기준선

> "적당히"는 없다. 아래 기준을 통과해야 완성. 모든 작업에 자동 적용됩니다.

## 릴스/숏폼 대본
- 첫 문장 3초 내 후킹 필수 (질문/충격/반전 중 하나)
- 전체 길이: 30~40초
- 모든 줄이 인사이트/꿀팁이어야 함 (인사·감사 filler 금지)
- CTA는 반드시 1개만 (여러 개 = 분산 = 전환율 하락)
- CEO 톤앤매너 3원칙 체크 완료 후 제출

## 보고서/분석
- 숫자는 반드시 데이터 출처 명시
- 구조: **요약 → 핵심 인사이트 3개 이내 → 액션 아이템**
- 액션 아이템에 담당자/기한 없으면 미완성

## 강의/교육 기획
- 한 모듈 = 한 가지 변환 (Before → After 명확)
- "수강생이 이걸 듣고 뭘 할 수 있게 되는가" 한 줄로 답변 가능해야 함
- 실습 비중 50% 이상 (이론만 나열 금지)

## 전략/기획 문서
- 문제 정의 → 선택지 비교 → 추천안 + 근거 구조 필수
- "좋을 것 같습니다" 류 모호한 결론 금지
- 구체적 수치/기한 포함

## 모든 산출물 공통
- 추측/추론 금지 (Zero-Inference)
- 데이터 없으면 "데이터가 필요합니다"라고 답변
- 확신도(%) 표기 (특히 전략 판단)
