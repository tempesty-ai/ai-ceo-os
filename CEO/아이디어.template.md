# 아이디어 박스

> 떠오르는 사업 아이디어를 한 줄씩 메모합니다.
> 김이사에게 "이거 검토해줘" 하면 즉시 분석해줍니다.

## 형식

```
- [{{날짜}}] {{한 줄 아이디어}} (확신도: {{0~100%}})
```

## 인큐베이팅 (검토 필요)

- [{{날짜}}] {{아이디어 1}} (확신도: 50%)
- [{{날짜}}] {{아이디어 2}} (확신도: 30%)

## 검증 중 (MVP 진행 중)

- [{{날짜}}] {{검증 중인 아이디어}} → 결과 예정: {{날짜}}

## 채택 (실행 중)

- [{{날짜}}] {{채택된 아이디어}} → 출시일: {{날짜}}

## 폐기

- [{{날짜}}] {{폐기된 아이디어}} — 폐기 사유: {{사유}}
