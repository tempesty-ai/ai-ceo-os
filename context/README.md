# Context (사업 맥락)

> AI 에이전트가 당신의 사업을 이해하기 위한 핵심 문서입니다.
> 이 폴더의 파일들이 AI의 "상식"이 됩니다.

---

## 필수 파일 3개

### 1. ceo-persona.md — CEO 톤앤매너
AI가 만드는 모든 콘텐츠에 적용되는 화법/스타일 정의.

```markdown
# CEO 페르소나

## 톤앤매너 3원칙
1. {{원칙 1}} — 예: "친근한 과외 선생님" 톤
2. {{원칙 2}} — 예: 즉시 후킹
3. {{원칙 3}} — 예: 전문용어 후 "쉽게 말하면..."

## 화법 패턴
- 오프닝: {{자주 쓰는 첫 문장 패턴}}
- 클로징: {{자주 쓰는 마무리 패턴}}
- 금지: {{절대 하면 안 되는 화법}}
```

### 2. mission-vision.md — 미션/비전/전략
사업의 방향성. AI가 판단할 때 이 방향과 정합하는지 확인합니다.

### 3. channel-rules.md — 채널별 규칙
인스타/유튜브/블로그 등 각 채널의 운영 규칙.
