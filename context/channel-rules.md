## 인스타그램 운영 규칙

### 콘텐츠 전략
- **주제 2대 축**: AI+육아 / AI+인사이트
- **콘텐츠 비율**: AI+인사이트 50% / AI+육아 50%
- **포맷 우선순위**: 릴스 > 캐러셀 > 단일 이미지
- **업로드 주기**: 주 3~5회 (초기 알고리즘 신뢰 구축)

---

### 릴스 규칙

- **길이**: 15~30초 (초기 채널 — 짧게 완주율 확보)
- **첫 3초**: 반드시 후킹 (질문/숫자/반전 중 하나) — 텍스트는 0.5초 이내 화면 등장
- **CTA**: 1개만 ("저장", "댓글", "팔로우" 중 택1)
- **자막**: 필수 (무음 시청자 50% 이상)
- **해시태그**: 3~5개 (니치 특화 — 과도한 해시태그 금지)
- **텍스트 위치**: 상단 or 중앙 1/3 — 하단 20% 핵심 텍스트 절대 금지 (인스타 UI 가림)

#### 릴스 대본 구조 (3가지 선택)

**구조 A — 문제→해결책 (신뢰 구축에 강함)**
```
훅: 문제 제시 → 문제 구체화(공감) → 해결책 → 결과 → CTA
```

**구조 B — 카운트다운 리스트 (저장 유도에 강함)**
```
훅: "딱 N가지" → 항목 N(약) → ... → 항목 1(강) → CTA
```
> 가장 강한 것을 마지막에 → 완료율 상승

**구조 C — 반전 서사 (팔로우 전환에 강함)**
```
훅: 일반 통념 → "하지만 실제로는..." → 반전 → 이유 → CTA
```

---

### 캡션 규칙

- **첫 줄**: 후킹 문장 (접기 전에 보이는 줄) — 릴스와 다른 각도로 작성
- **구조**: 후킹 → 본문 가치(2~3줄) → CTA
- **이모지**: 포인트용으로만 (과다 사용 금지)
- **CTA**: 1개만 (저장 or 댓글 or 팔로우 — 중복 금지)

#### 캡션 템플릿

```
[후킹 1줄] — "설마 아직도...?" 또는 숫자 충격형

[핵심 가치 2~3줄]
- 현직 APM 엔지니어로서 실제로 써보고 검증한 것
- 즉시 써먹을 수 있는 구체적 내용

[키워드 줄] — 검색 가능한 용어 자연스럽게 포함
예: AI 육아 / AI 활용법 / ChatGPT 프롬프트

[CTA 1개만]
예: "저장해두고 필요할 때 꺼내보세요"
또는 "댓글에 [키워드] 남기면 자료 보내드릴게요"

[해시태그] 3~5개 (니치 특화)
예: #AI활용 #AI육아 #ChatGPT활용법
```

#### 캡션 금지 행동

- 저장 + 공유 + 댓글 + 팔로우 동시 요청 (아무것도 안 하게 됨)
- "이번 릴스에서는..."으로 캡션 시작
- 해시태그 20개 이상

---

### 알고리즘 공식 (우선순위)

```
트렌드 + CTA + 상단카피 + 후킹 = 도달 극대화

요소별 점수:
- 트렌드 반영   → 10/10
- CTA           → 10/10
- 상단 카피     → 9/10
- 후킹          → 8/10
- 해시태그      → 0/10 (조회수와 무관)
- 1일 1포스팅   → 0/10 (질이 양보다 중요)
```

---

### DM 루프 설계 (댓글 → DM → 재방문)

1. 릴스 CTA: "댓글에 [키워드] 남겨주세요"
2. 댓글 감지 → DM으로 자료/링크 자동 발송 (Manychat 또는 수동)
3. DM 내 링크트리 → 강의/채널 추가 방문 유도
4. 자료 받은 팔로워 → 다음 릴스 알람 설정 유도

> 댓글 알고리즘 부스트 효과 + 팔로워 전환율 동시 확보

### 성과 KPI 기준선 (30일차 판단 기준)

| 지표 | 합격 기준 | 의미 |
|:---|:---:|:---|
| 저장률 | 2%+ | 콘텐츠 장기 가치 — 이 포맷 2배 확장 |
| 완료율 | 70%+ | 훅+구조 작동 여부 |
| 팔로우 전환율 | 0.5%+ | 니치 명확성 |
| 공유율 | 1%+ | 입소문 |

> 30일 후: 저장률 상위 3개 분석 → 2개월차에 그 포맷 두 배로 늘리기
> 데이터 없으면 판단 금지 (Zero-Inference Rule)

---

### 금지 행동

- 검증되지 않은 육아 의학 정보 게시 금지
- 어뷰징성 팔로우/언팔 반복 금지
- 타 계정 무단 콘텐츠 도용 금지
- 과장·허위 성과 클레임 금지
