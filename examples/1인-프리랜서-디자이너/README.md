# 예시: 1인 프리랜서 디자이너 — 완성본 세트

> **이게 "채워지면 이렇게 됩니다" 샘플입니다.**
> ai-ceo-os를 clone한 직후 `{{slot}}`이 가득한 빈 상태에서, `/setup`을 완료하고 한 달 운영한 후의 모습을 미리 보여줍니다.

---

## 가상 페르소나

- **이름**: 박소영 (가명)
- **업**: 1인 브랜드 디자이너 (로고, 패키지, 상세페이지 전문)
- **경력**: 대기업 인하우스 5년 → 프리랜서 3년차
- **매출**: 월 평균 400~600만원 (프로젝트 3~5건)
- **고민**:
  - 포트폴리오로 문의는 오는데 **견적 단계에서 이탈이 많음**
  - 인스타 팔로워 1.2만인데 **월 문의가 10건 미만**
  - "다음 달 수입이 얼마일지" 예측이 안 됨
  - 1인이라 세일즈/마케팅/제작을 혼자 다 함 → 시간 부족

---

## 이 폴더가 담고 있는 것

| 파일 | 설명 |
|:---|:---|
| `CLAUDE.md` | 박소영의 사업 정의, 고객 여정 6단계, 금지 행동, 자주 시키는 작업 Top 3 |
| `context/ceo-persona.md` | 박소영의 톤앤매너 — "친근한 선배 디자이너" |

| `tasks/todo.md` | 실제 운영 중인 할 일 (이번 주/이번 달) |
| `tasks/lessons.md` | 한 달 운영하며 쌓인 교훈 3건 |
| `tasks/predictions.md` | 진행 중인 전략 예측 2건 |
| `skills/portfolio-post/SKILL.md` | 포트폴리오 포스트 작성 전용 커스텀 스킬 |

## 이 샘플에서 배울 점

1. **빈 템플릿이 이렇게 채워진다** — `{{업종}}`, `{{고객 여정}}` 같은 추상 슬롯이 박소영의 **구체적 현실**로 변환된 형태
2. **C-Suite 9명이 어떻게 커스텀되는가** — 9명 전부가 아닌, **1인 프리랜서에게 필요한 3명(CMO, CPO, CXO)에 먼저 집중** (CFO는 월 1회만, CTO는 필요 없음)
3. **직접 호출 없이 AI가 일하는 방식** — 박소영이 "이번 주 포트폴리오 뭐 올리지?" 치면 김이사가 CCO 소집 → 3개 초안 → 박소영 선택
4. **교훈이 실제로 쌓이는 모습** — 추상적 원칙이 아니라, "2026-03-15 — 견적서 첨부 없이 단가만 DM으로 보냈더니 이탈 70% → 교훈: 첫 응답에 항상 패키지 PDF 첨부"

## 사용법

### 1. 그대로 실행해보기
```bash
cp -r examples/1인-프리랜서-디자이너/* .
# 주의: 이러면 당신의 기존 CLAUDE.md가 덮어써집니다
```

### 2. 읽기만 하기
박소영의 파일을 **참고만** 하고, 본인 사업에 맞춰 `/setup`을 돌리세요.

### 3. 수정해서 다른 업종으로 확장
이 구조를 그대로 따라 "1인 코칭", "카페 운영자", "쇼핑몰 판매자" 등 본인 업종 예시를 만들어 PR로 기여하세요.
