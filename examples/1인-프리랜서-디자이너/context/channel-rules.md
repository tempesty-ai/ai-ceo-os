## Instagram 운영 규칙

### 발행 주기
- **주 5회** (월/화/목/금 + 토요일 스토리)
- 월: 포트폴리오 포스트 (before/after 스토리)
- 화: 릴스 (교육형 — "1인 사업자를 위한 브랜딩 팁")
- 목: 릴스 (바이럴 — 논쟁형/반전형)
- 금: 카드뉴스 (체크리스트형)
- 토: 스토리 (작업 과정, 비하인드)

### 해시태그 규칙
- **포트폴리오 포스트**: 5~8개 (#브랜드디자인 #로고디자인 #상세페이지 #1인사업자브랜딩 등)
- **릴스 교육형**: 3~5개
- **릴스 바이럴형**: 0~2개 (알고리즘이 해시태그 없는 게 더 잘 밀어줌, 2026년 기준)
- **카드뉴스**: 5~7개

### 금지 해시태그
- `#맞팔`, `#선팔`, `#팔로우백`, `#데일리`, `#일상` (스팸 취급)
- 지역 해시태그 (박소영은 지역 고객 아님)

### 프로필 링크
- 포트폴리오 사이트 1개 고정
- 이벤트 있으면 linktr.ee로 변경 (이벤트 종료 후 즉시 복귀)

## CTA 규칙

각 채널/콘텐츠 유형별로 **CTA는 정확히 1개**만.

| 콘텐츠 | 1순위 CTA | 금지 CTA |
|:---|:---|:---|
| 릴스 교육형 | "저장해두세요" | "팔로우 부탁" (직접 구걸 금지) |
| 릴스 바이럴형 | "공유해 주세요" | "좋아요 눌러주세요" (레거시) |
| 포트폴리오 포스트 | "프로필 링크 → 무료 진단 체크리스트" | "DM 주세요" (상담 프로세스 없이 유입되면 관리 불능) |
| 카드뉴스 | "저장 후 체크리스트로 사용" | 복수 CTA |
| 스토리 | 스티커 1개 (투표/질문/링크 중 택 1) | 없음 |

## 금지 콘텐츠 유형

1. **경쟁 디자이너 비교**: 프리랜서 커뮤니티는 좁다. 적 만들지 마라.
2. **AI 이미지 결과물 과시**: "챗GPT로 만든 로고" 같은 콘텐츠 — 포지셔닝 훼손
3. **수강생/강의 유도**: 박소영은 교육 사업이 아님
4. **정치/종교/젠더 이슈**: 어떤 경우에도 금지
5. **가격 노출 콘텐츠**: 가격은 상담에서만 공개
6. **과장된 결과 약속**: "매출 10배!" 류 금지

## 포트폴리오 사이트 규칙

- **첫 화면**: 3초 내 "이 사람이 뭘 하는지" 답 (포지셔닝 문구 + 대표 사례 3개)
- **사례 구조**: 각 사례는 반드시 `문제 → 접근 → 결과(숫자)` 3단 구조
- **연락 수단**: "30분 무료 상담 예약" 1개만 (이메일/전화 노출 X)
- **가격**: 노출 X. "상담 후 견적" 안내만.
- **후기**: 허위 후기 절대 금지. 실제 고객 동의 받은 것만.

## 뉴스레터 규칙 (월 1회)

- **발행일**: 매달 첫째 월요일 오전 9시 (고정)
- **구성**:
  1. 이번 달 케이스 스터디 1개 (전/후 + 인사이트)
  2. 1인 사업자를 위한 팁 1개
  3. 박소영 이번 달 배운 것 1개
  4. 진행 가능한 프로젝트 슬롯 안내 (월 N건)
- **금지**: 할인/이벤트 남발. 신뢰 채널이라 세일즈 남발하면 구독 취소.

## 위기 대응 (우발 상황)

### 악플/트집
- 첫 번째: 무시하지 말고 **공개적으로 정중히 답변** (다른 사람들이 본다)
- 두 번째: 같은 사람 반복 시 블록 + 캡처 보관
- 정치/인격 공격: 즉시 블록 + 신고

### 프로젝트 지연
- 고객에게 **사전 고지** (마감 1주 전)
- 일정 재협의, 이유 정직하게 (거짓말 금지)

### 결제 분쟁
- 모든 계약은 **이메일 + 계약서 PDF** 기록
- 구두 합의 금지
