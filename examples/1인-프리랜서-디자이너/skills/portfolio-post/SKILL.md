# 포트폴리오 포스트 작성 스킬 (박소영 전용)

> "완성 이미지 올리기" 수준이 아니라, **"이 사람이 내 문제도 풀어줄까?"**를 증명하는 구조.

CCO 서브에이전트로 전환. CMO는 검증 단계에서만 소집.

## 2단계: 작업 선택

포트폴리오에 올릴 **완료된 프로젝트 1건**을 사용자에게 받습니다. 없으면 역질문:

```
어떤 프로젝트 올릴까요?
- 가장 결과가 좋았던 것 (매출 변화 숫자 있는 것 우선)
- 최근 3개월 내 완료된 것
- 클라이언트 공개 동의 받은 것
```

## 3단계: "문제 → 접근 → 결과" 3단 구조 작성 (강제)

### Section 1: 문제 (Problem)

- **헤드라인**: "[고객 업종] 대표님의 진짜 고민"
- **내용**:
  - 의뢰 전 상태 (브랜딩/매출/이미지 등)
  - "OO인데 왜 매출이 안 날까요?" 같은 **고객 목소리 직접 인용**
  - 숫자: 의뢰 전 월 매출 또는 문제 지표
- **금지**: "로고가 예쁘지 않았다" 같은 디자이너 시각. 반드시 **고객 고통** 중심.

### Section 2: 접근 (Approach)

- **헤드라인**: "제가 가장 먼저 한 건 [핵심 접근] 이었어요"
- **내용**:
  - 해결 전략 1줄 (박소영의 관점)
  - 구체적 디자인 결정 3가지 (폰트/컬러/구조 등)
  - **"왜 이렇게 했는지" 이유**
- **금지**: 디자인 전문용어 폭탄. 반드시 "쉽게 말하면..." 풀어쓰기.

### Section 3: 결과 (Result)

- **헤드라인**: "3개월 후"
- **내용**:
  - **반드시 숫자**: 매출 변화 / 문의 증가 / 전환율 개선 등
  - 고객 반응 1줄 인용
  - Before/After 이미지 2장
- **금지**: "대표님도 만족하셨어요" 같은 공허한 문장. 숫자 없으면 작성 포기.

## 4단계: 캡션 + CTA

```markdown
## 캡션 형식

[1줄 후킹 — "월 매출 800만원 → 1,400만원. 브랜드만 바꿨을 뿐인데."]

[2~3줄 본문 — 문제의 핵심과 접근의 이유]

[CTA — 정확히 1개]
프로필 링크 → 무료 브랜드 진단 체크리스트 10문항

해시태그: #브랜드디자인 #1인사업자브랜딩 #로고디자인 #상세페이지디자인 #브랜딩전략 (5~8개)
```

---

## 5단계: 이미지 구성 (CCO 기획)

슬라이드 캐러셀 기준 5장 권장:

| # | 내용 | 메모 |
|:---|:---|:---|
| 1 | 후킹 타이틀 (1줄 카피) | 첫 3초에 이탈 안 시키는 문구 |
| 2 | 문제 — 의뢰 전 상태 | Before 이미지 or 고객 문제 인용 |
| 3 | 접근 — 핵심 결정 3가지 | 텍스트 슬라이드 |
| 4 | 결과 — Before/After 대비 | 숫자 강조 (매출 변화) |
| 5 | CTA — 무료 진단 안내 | 프로필 링크 유도 |

---

## 6단계: CMO 검증 (전환 관점)

CMO 서브에이전트에 독립 검증 의뢰:

```python
Task(
    subagent_type="cmo",
    description="포트폴리오 포스트 전환 관점 검증",
    prompt="""
    [포스트 초안]
    {cco_output}

    체크:
    1. CTA가 정확히 1개이고 "30분 무료 상담"으로 이어지는가
    2. 가격 노출이 있는가 (있으면 감점 — channel-rules.md 참조)
    3. 숫자 기반 결과 증명이 Section 3에 포함되었는가
    4. 고객 여정 단계 2~3(신뢰 구축) 톤과 맞는가

    점수 (1~5) + 개선 제안 최대 3개.
    """
)
```

## 이 스킬이 실패하는 3가지 시나리오 (Inversion)

1. **숫자 없이 "감"으로 쓰기** → 조기 경고: Section 3에 숫자 없으면 즉시 재작성
2. **CCO가 박소영 톤 이탈 (너무 전문가스럽게)** → 조기 경고: "~입니다" 어미 3개 이상이면 재작성
3. **디자이너 자기만족 ("폰트 선택의 의미")** → 조기 경고: 고객 고통 언급이 1개 미만이면 재작성
