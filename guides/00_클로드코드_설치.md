# Claude Code 설치 가이드

> 처음부터 끝까지 따라하면 10분 안에 완료됩니다.
> Windows와 Mac 모두 지원합니다.

---

## 전체 흐름

```
Step 1: Node.js 설치 (3분)
  ↓
```

## Step 1: Node.js 설치

Claude Code는 Node.js 위에서 돌아갑니다. 먼저 Node.js를 설치하세요.

### Windows

1. https://nodejs.org 접속
2. **LTS (추천)** 버튼 클릭 → 다운로드
3. 설치 파일 실행 → 전부 **Next** 클릭 → **Install**
4. 설치 완료 후 **터미널**(명령 프롬프트 또는 PowerShell) 열기
5. 확인:
   ```
   node --version
   ```
   `v20.x.x` 또는 `v22.x.x`가 나오면 성공!

### Mac

1. https://nodejs.org 접속
2. **LTS (추천)** 버튼 클릭 → 다운로드
3. `.pkg` 파일 실행 → 설치
4. **터미널** 열기 (Spotlight에서 "터미널" 검색)
5. 확인:
   ```
   node --version
   ```
   `v20.x.x` 또는 `v22.x.x`가 나오면 성공!

> **이미 Node.js가 설치되어 있다면** 이 단계는 건너뛰세요.
> `node --version`을 입력해서 버전이 나오면 이미 설치된 것입니다.

---

## Step 2: Claude Code 설치

터미널에서 아래 명령어를 입력하세요:

```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:
```bash
claude --version
```

버전 번호가 나오면 설치 완료!

> **에러가 나는 경우:**
> - Windows: PowerShell을 **관리자 권한**으로 실행 후 다시 시도
> - Mac: `sudo npm install -g @anthropic-ai/claude-code` 입력 (비밀번호 입력)

---

## Step 3: API 키 발급 + 인증

Claude Code를 사용하려면 Anthropic API 키가 필요합니다.

### 3-1. API 키 발급

1. https://console.anthropic.com 접속
2. 회원가입 (구글 계정으로 가능)
3. 좌측 메뉴에서 **API Keys** 클릭
4. **Create Key** 버튼 클릭
5. 이름 입력 (예: "my-ai-os") → **Create**
6. 생성된 키를 **복사** (sk-ant-로 시작하는 긴 문자열)

> ⚠️ 이 키는 한 번만 보여줍니다! 반드시 안전한 곳에 저장하세요.

### 3-2. Claude Code 인증

터미널에서:

```bash
claude
```

처음 실행하면 인증 방법을 물어봅니다.
**API Key** 방식을 선택하고, 복사한 키를 붙여넣으세요.

> **요금 안내:**
> - API 사용량에 따라 과금됩니다
> - 일반적인 사용 시 월 $5~20 수준
> - https://console.anthropic.com 에서 사용량 확인 가능
> - 크레딧을 미리 충전하거나 결제 수단을 등록해야 합니다

## Step 5: AI CEO OS 셋업

### 5-2. Claude Code 실행

```bash
claude
```

### 5-3. 셋업 시작

Claude Code 안에서:
```
/setup
```

AI가 질문을 시작합니다. 답변하면 자동으로 AI CEO OS가 구축됩니다!

## 문제 해결 (FAQ)

### "command not found: claude"
→ Node.js 설치 후 터미널을 **껐다가 다시 열어야** 합니다.

### "npm ERR! permission denied"
→ Windows: 관리자 권한 PowerShell 사용
→ Mac: `sudo` 붙이기

### "API key invalid"
→ 키를 다시 복사해서 붙여넣기. 앞뒤 공백이 없는지 확인.

### "크레딧 부족"
→ https://console.anthropic.com 에서 크레딧 충전 또는 결제 수단 등록

### Claude Code가 영어로 답해요
→ 첫 메시지에 "한국어로 대화해줘"라고 입력하면 됩니다.
→ 또는 CLAUDE.md에 `언어: 한국어`가 설정되어 있으면 자동으로 한국어 사용.
