# 커스터마이징 가이드

---

## 에이전트 커스터마이징

### C-Suite 구조는 유지, 내용은 당신 것으로

이 시스템의 핵심은 **8명의 C-Suite 에이전트**가 서로 다른 관점에서 판단하는 구조입니다.

구조(CSO → C-Suite 8명)는 그대로 두고, 각 에이전트의 **판단 기준**만 당신의 사업에 맞게 바꾸세요.

예시 — 커피숍 운영자라면:
- CMO의 핵심 질문: "이 프로모션이 재방문율을 높이는가?"
- CPO의 핵심 질문: "이 메뉴가 원가율 30% 이내인가?"
- CXO의 핵심 질문: "고객이 15분 안에 음료를 받는가?"

### 팀원 추가

C-Suite 아래에 전문 팀원을 추가할 수 있습니다.

```markdown
# agents/cmo/social-media-manager.md

## 역할
소셜 미디어 운영. 인스타/유튜브/틱톡 채널 관리.

## 전문 분야
- 콘텐츠 일정 관리
- 댓글/DM 응대
- 해시태그 전략
```

## 스킬 추가

스킬 = 반복되는 업무의 매뉴얼입니다.

### 스킬 작성 구조

```markdown
# skills/[카테고리]/[스킬명].md

## 목적
이 스킬이 해결하는 문제 (1줄)

## 입력
무엇이 필요한가

## 프로세스
1. Step 1
2. Step 2
3. Step 3

## 출력
무엇이 나오는가

## 품질 기준
완성의 기준은 무엇인가
```

## 워크플로우 설계

워크플로우 = 여러 에이전트가 협업하는 파이프라인입니다.

### 3-Phase 구조 (권장)

```
Phase 1: 리서치 + 자산 로딩 (병렬)
  ↓
Phase 2: 통합 + 초안 작성
  ↓
Phase 3: 검증 (병렬)
  ↓
최종 납품
```

이 구조를 당신의 업무에 적용하세요:

- **콘텐츠 제작**: 리서치 → 초안 → 편집 검증
- **상품 기획**: 시장 분석 → 기획서 → 재무/기술 검증
- **고객 응대**: 문의 분류 → 답변 작성 → 품질 검토

## 자주 묻는 질문

**Q: 에이전트를 8명 다 만들어야 하나요?**
A: 아니요. 1~2명부터 시작하세요. 필요할 때 추가하면 됩니다.

**Q: Claude가 아닌 다른 AI로도 가능한가요?**
A: 구조 자체는 AI 모델에 독립적입니다. CLAUDE.md를 해당 AI의 프로젝트 설정에 맞게 변환하면 됩니다.

**Q: 1인 기업인데 C-Suite이 필요한가요?**
A: 1인이니까 더 필요합니다. 혼자서 마케팅, 재무, 콘텐츠, 고객관리를 다 하잖아요. AI가 각 역할의 관점에서 판단해줍니다.
