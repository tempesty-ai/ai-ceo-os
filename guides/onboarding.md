## 🚀 D+0 (오늘 30분) — 첫 임원 호출 1회

**미션**: **김이사**한테 "오늘 내가 뭘 해야 해?" 물어보기.

```
나: 김이사, 내 사업 보고 오늘 가장 먼저 해야 할 일 1개만 골라줘.
```

→ 김이사가 당신의 `CLAUDE.md`를 읽고 1개 골라줍니다.
→ **이게 첫 성공 체험입니다.** 그대로 따라하세요.

**✅ 인증**: 디스코드 `#첫미션` 채널에 "김이사가 [X]를 하라고 했어요" 한 줄 인증.

## 📅 D+1 ~ D+3 — 임원 9명 모두 인사

매일 3명씩 호출해서 인사 + 본인 사업 분석 1건 요청:

```

```

→ 9명을 다 만나면 **누가 어떤 일을 잘하는지** 감이 옵니다.

## 📆 Week 1 — 첫 Tier 2 회의 1회

당신 사업의 가장 큰 고민 1개를 김이사에게 던지세요:

```
나: 김이사, 우리 [고민]을 해결해야 해. 임원회의 한번 돌려줘.
```

→ 김이사가 관련 임원 2~3명을 자동 소집해서 분석합니다.
→ **이게 진짜 "AI 팀이 일하는" 순간입니다.**

## 🗓 Month 1 — 본격 커스터마이징

이 시점부터는 임원을 **당신 사업에 맞춰** 진화시키세요:

1. 각 임원 `agents/{직책}/README.md` 열어서 **"내면 참모진 4명"** 빈칸 채우기
2. 각 임원의 **MUST Disagree 4조건**을 본인 사업 맥락으로 작성
3. `tasks/lessons/{직책}.md`에 매일 1개씩 교훈 누적
4. `tasks/predictions.md`에 김이사 전략 판단을 P-001부터 기록

→ 한 달 후, 9명의 임원이 **당신만의 임원**이 됩니다.
