# outputs/ 산출물 폴더 가이드

> AI가 만든 모든 결과물은 `outputs/`에 자동 저장됩니다.

## 구조

```
outputs/
└── YYYY/
    └── MM/
        └── DD/
            ├── [릴스]_제목.md
            ├── [유튜브]_제목.md
            └── [분석]_제목.md
```

## 규칙

- **경로**: `outputs/{연도}/{월}/{일}/[유형]_제목.md`
- **유형 태그**: `[릴스]` `[유튜브]` `[전략]` `[기획]` `[분석]` `[강의]` `[세일즈]` `[회고]` `[기타]`
- 상세: `.claude/rules/산출물.md`

## 사용법

1. AI에게 작업 지시 ("이번 주 릴스 대본 써줘")
2. AI가 결과물 생성
3. AI가 자동으로 `outputs/2026/04/07/[릴스]_..._대본.md`에 저장
4. CEO가 나중에 찾을 때 날짜 또는 태그로 검색

## 검색 팁

```bash
# 4월 릴스 모두 보기
ls outputs/2026/04/*/[*릴스*]*

# 분석 보고서 모두 보기
find outputs -name "[분석]_*"
```

## 왜 이 구조인가

1. **시간순 자동 정렬**: 오래된 산출물 찾기 쉬움
2. **태그로 분류**: 즉시 grep 가능
3. **AI 자동 저장**: 김이사가 결과물 만들면 이 경로로 자동 저장
4. **백업 안전**: 날짜별 폴더는 백업 시스템과 잘 맞음
