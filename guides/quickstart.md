# 퀵스타트 가이드

> 처음부터 끝까지 따라하면 15분 안에 나만의 AI CEO OS가 완성됩니다.

---

## 전체 흐름

```
Step 0: Claude Code 설치 (10분, 최초 1회)
  ↓
Step 1: 프로젝트 준비 (2분)
  ↓
Step 2: /setup 실행 (5분)
  ↓
Step 3: 첫 번째 업무 시켜보기 (1분)
```

---

## Step 0. Claude Code 설치

Claude Code가 아직 설치되어 있지 않다면:

→ **[Claude Code 설치 가이드](00_클로드코드_설치.md)** — Windows/Mac 모두 지원

이미 설치되어 있다면 Step 1로 바로 가세요.

---

## Step 1. 프로젝트 준비

### 1-1. 파일 준비

**방법 A — ZIP 받으신 경우 (현재 기본 배포 방식)**
1. 받은 `ai-ceo-os-v1.0.zip` 압축 해제
2. 원하는 위치로 폴더 이동 (예: `~/projects/ai-ceo-os`)

**방법 B — GitHub에서 clone (공개 후)**
```bash
git clone https://github.com/AINOW-official/ai-ceo-os.git
cd ai-ceo-os
```

### 1-2. 해당 폴더로 이동

```bash
cd ai-ceo-os
```

### 1-2. Claude Code 실행

```bash
claude
```

---

## Step 2. /setup 실행 (핵심!)

Claude Code 안에서 아래를 입력하세요:

```
/setup
```

AI가 당신의 사업에 대해 질문합니다. 답변만 하면 됩니다.

**5~10분 후 자동으로 생성되는 것:**
- `CLAUDE.md` — AI의 두뇌 (당신의 사업 정보)
- `context/ceo-persona.md` — 당신의 말투와 스타일
- `context/orchestration.md` — AI 팀 협업 규칙
- 에이전트 3명 커스터마이징 — 당신의 사업에 맞는 판단 기준
- 첫 번째 스킬 — 가장 시간 많이 드는 업무의 AI 매뉴얼
- `tasks/` — 작업 관리 + 학습 기록

> **"모르겠어요"도 유효한 답입니다.** AI가 업종에 맞는 기본값을 자동 적용합니다.

---

## Step 3. 첫 번째 업무 시켜보기

셋업이 끝나면 AI가 첫 번째 미션을 안내합니다.
바로 시켜보세요!

```
예: "이번 주 인스타 콘텐츠 3개 기획해줘"
예: "신규 고객 유입 전략 짜줘"
예: "이번 달 매출 목표 달성 계획 세워줘"
```

→ AI가 당신의 CLAUDE.md를 참고해서 **당신의 사업에 맞는** 답변을 합니다!

> 🚀 **여기까지 했다면 → [`onboarding.md`](onboarding.md)로 이동하세요.**
> 첫 30분 / 첫 1주 / 첫 1달 동안 무엇을 해야 하는지 시간축 가이드입니다.
