# Knowledge (지식 저장소)

> AI 에이전트가 판단할 때 참고하는 실제 데이터와 전략 자료입니다.
> 당신의 데이터를 여기에 넣으세요.

---

## 폴더 구조

```
knowledge/
├── channel-data/         ← 채널 성과 데이터
│   ├── instagram/        ← 인스타 인사이트, 릴스 성과
│   ├── youtube/          ← 유튜브 분석, 영상별 데이터
│   └── ads/              ← 광고 성과 데이터
│
└── strategy-references/  ← 전략 참고 자료
    └── (전략서 요약, 시장 분석 등)
```

## 넣어야 할 것

### channel-data/
- **인스타**: 릴스별 조회수, 저장 수, 공유 수, 팔로워 유입 경로
- **유튜브**: 영상별 조회수, 시청 지속 시간, 구독 전환
- **광고**: 캠페인별 ROAS, CPA, 전환율

### strategy-references/
- 참고하는 경영/마케팅 서적의 핵심 요약
- 경쟁사 분석 자료
- 시장 트렌드 보고서

## 왜 필요한가

AI 에이전트가 "데이터 기반 판단"을 하려면 실제 데이터가 있어야 합니다.
데이터 없이 추론하면 틀립니다. **Zero-Inference Rule**: 데이터가 없으면 "데이터가 필요합니다"라고 답하는 것이 정답.
