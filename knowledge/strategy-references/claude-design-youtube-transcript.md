## 핵심 요약

Anthropic이 출시한 **Claude Design** (리서치 프리뷰) 기능 소개.
디자인 시스템 → 와이어프레임 → 프로토타입 → 슬라이드덱 → Claude Code 핸드오프까지
**전체 풀스택 디자인 파이프라인을 하나의 컨텍스트 안에서** 처리 가능.

---

## 주요 기능

### 1. 디자인 시스템 셋업
- 회사 정보, GitHub 링크, 소스코드, Figma 파일 입력
- 액센트 컬러, 아바타, 배치 등 자동 생성
- 좌측 채팅으로 요소 변경 가능

### 2. 와이어프레임 생성
- 타입 선택 (와이어프레임 / 슬라이드덱)
- 프롬프트 입력 → Claude가 의도 파악 질문
  - 디자인 유형, 플랫폼, 타겟, 페이지 구성, 시안 수, 변수 항목
- 여러 시안 동시 생성 (레이아웃 구조, 피드백 형태, 내비게이션 방식 등 차이)
- 변수 옵션: 다크/라이트 모드, 열 수, 밀도, 레이아웃 등

### 3. 프로토타입 제작
- 와이어프레임에서 옵션 선택 후 "완전한 프로토타입 만들어줘" 요청
- 실제 버튼 작동, 다이얼로그 등 인터랙션 포함
- Edit 버튼으로 특정 요소 직접 수정 가능
- 코멘트 기능으로 특정 요소만 프롬프트 수정

### 4. 슬라이드덱 생성
- "디자인한 웹사이트 소개 슬라이드덱 만들어줘" 요청
- 제작한 디자인 기반으로 자동 프레젠테이션 생성
- Edit으로 수정 가능

### 5. Claude Code 핸드오프
- Export → Claude Code 핸드오프 선택
- 커맨드 복사 후 Claude Code에 붙여넣기
- 디자인 시스템 자동 참조하여 개발 즉시 가능

---

## 차별화 포인트

- **하나의 컨텍스트**: 스티치, 페이퍼, 펜슬 등 기존 툴과 달리 전체 파이프라인이 단일 컨텍스트 내에서 처리
- **주관적 의견 반영**: 단순 생성 도구가 아닌 Anthropic의 디자인 철학이 녹아있음
- **파일 시스템**: 생성물이 디자인 파일 폴더에 자동 저장 및 관리
