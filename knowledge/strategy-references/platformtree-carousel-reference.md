```
슬라이드 1: 부정 후킹 ("~하지 마세요")

슬라이드 5: 혜택 3가지 (숫자 포함)
슬라이드 6: 댓글 CTA → DM 리드 수집
```

---

## 왜 이 구조가 먹히는가

- **"하지 마세요"** — 반전 후킹, 스크롤 멈춤
- **댓글 CTA** — 알고리즘 참여율 UP + DM 리드 자동 수집 동시
- **혜택을 숫자로** ("5시간 이상") — 구체적 = 신뢰
- **리드 마그넷** — 댓글 달면 뭔가 받는다는 행동 유도
