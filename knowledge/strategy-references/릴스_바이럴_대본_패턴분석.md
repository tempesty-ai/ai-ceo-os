# 릴스 바이럴 대본 패턴 분석

> 출처: 실제 바이럴 릴스 대본 수집 분석  
> 활용: 릴스 대본 작성 시 후킹·구조·CTA 참고용

---

## 핵심 후킹 공식 (검증된 오프닝 패턴)

### 패턴 A — 도전/내기형 (긴장감 즉시 생성)
```
"과연 [캐릭터]가 [기간] 안에 [목표]를 달성할 수 있을까요?"
"만약 [기간] 안에 [목표]를 달성하지 못한다면 이 영상은 삭제됩니다."
```
> 후킹 원리: 결과가 궁금해서 끝까지 보게 됨. 실패 가능성 언급이 긴장감 유발.

### 패턴 B — 금지/경고형 (역설 클릭 유발)
```
"그냥 이거 모르면 [X] 쓰지 마세요. 아무 소용없습니다."
"하... 제발 이거는 꼭 알고 [X] 사용하셨으면 좋겠어요."
"[X] 안 쓰면 올해도 돈 못 법니다."
"설마 아직도 [구식 방법] 하시는 건 아니시죠?"
```
> 후킹 원리: "나 해당되는데?" 자기검열 유발. 금지일수록 더 보고 싶어짐.

### 패턴 C — 숫자 충격형
```
"[X]% 아무도 모르는 [가치 있는 것] 알려드릴게요."
"한국인 90%가 모르는 [N]가지 기능"
"[N]분 만에 [놀라운 결과] 가능합니다."
"[금액] 아껴드릴게요."
```
> 후킹 원리: 구체적 숫자가 뇌에서 가장 빠르게 처리됨. 신뢰감+호기심 동시 유발.

### 패턴 D — 무료 증정형 (DM 유입 + 팔로우)
```
"이거 복사 붙여넣기만 하면 끝! 팔로우하고 댓글 남겨주시면 자료 즉시 보내드려요."
"팔로우하고 댓글에 [키워드] 남겨주시면 [자료] 바로 보내드릴게요."
```
> 후킹 원리: 즉각적 가치 + 행동 유도의 교환 공식. 댓글 알고리즘 부스트 효과.

### 패턴 E — 아무도 안 알려줬다형
```
"아무도 공개를 안 해서 제가 그냥 욕먹을 각오를 하고 다 풀어버릴게요."
"해외 전문가들 사이에서 유행하는 [N]개 알려드릴게요."
"[기관/대기업]에서 숨기고 있는 방법 알려드릴게요."
```
> 후킹 원리: 희귀 정보 + 희생감 → 신뢰 + 궁금증.

## AI 콘텐츠 고성과 주제 패턴

### 주제 유형별 후킹 공식

| 주제 | 검증된 후킹 문장 |
|:---|:---|
| AI 도구 무료화 | "설마 아직도 돈 주고 [AI] 구독하시는 건 아니시죠?" |
| GPT 꿀팁 | "그냥 이거 모르면 GPT 쓰지 마세요" |
| AI 이미지/영상 | "[N]분 만에 [놀라운 결과] 가능합니다" |
| 자동화 | "내가 자는 동안 [AI]가 자동화하는 방법" |
| 프롬프트 | "이거 복사 붙여넣기만 하면 끝" |
| AI 감지 우회 | "AI 감지기 99% 통과하는 방법" |
| 도구 비교 | "이거 말고 이거 쓰세요" (반복 리듬) |

## 대본 구조 공식 (실전 3단계)

### 구조 ① 문제 → 해결책 → 증거
```
[후킹] "아직도 [구식 방법] 하시는 건 아니시죠?"
↓
[문제 확대] "[구식 방법]을 쓰면 [불이익]합니다"
↓  
[해결책] "이것만 하면 됩니다. [구체적 도구/방법]"
↓
[증거] "실제로 이렇게 됩니다 [시연/결과]"
↓
[CTA] "팔로우하고 댓글로 [키워드] 남겨주세요"
```

### 구조 ② 리스트형 (저장 유도 최강)
```
[후킹] "[상황별] 딱 이거만 쓰면 끝"
↓
[항목 1] 상황 A → 도구 A
[항목 2] 상황 B → 도구 B
...
[마지막 항목] 가장 임팩트 있는 것
↓
[CTA] "저장해두고 필요할 때 써보세요"
```

### 구조 ③ 스토리텔링형 (공감 + 팔로우)
```
[충격 상황] "내 나이 [X], [예상 밖 상황]"
↓
[고난 묘사] 공감되는 현실적 어려움
↓
[전환점] "그때 [AI/도구/방법]을 알게 됐습니다"
↓
[결과] "[구체적 변화]가 일어났습니다"
↓  
[CTA] "과연 [나]의 운명은? 팔로우하면 다음 편 알람"
```

### 구조 ④ 단계별 튜토리얼형
```
[후킹] "[놀라운 결과] 바로 알려드릴게요"
↓
[Step 1] → [Step 2] → [Step 3]
(각 단계를 최대한 심플하게)
↓
[결과 시연]
↓
[CTA] "댓글로 [키워드] 남기면 프롬프트 보내드려요"
```

## CTA 공식 (행동 유도)

### 고성과 CTA 문장 모음
```
"팔로우하고 댓글에 [키워드] 남기면 자료 즉시 보내드려요"
"저장하고 나중에 꼭 써먹으세요"
"이거 필요한 분 계신가요? 댓글로 알려주세요"
"자세한 내용은 캡션에서 확인해주세요"
"복붙 가능한 프롬프트 보내드릴 테니 댓글 남기면 바로 드릴게요"
```

### CTA 유형별 선택 기준
| 목표 | CTA 유형 | 예시 |
|:---|:---|:---|
| 팔로워 증가 | 시리즈 예고 | "1탄은 이랬는데... 2탄에서는" |
| 저장률 증가 | 나중에 필요 강조 | "저장 안 하면 나중에 못 찾아요" |
| 댓글 증가 | 자료 증정 | "댓글에 [키워드] 남겨주세요" |
| 공유 증가 | 주변에 필요한 사람 | "이 영상 필요한 사람에게 공유해주세요" |
