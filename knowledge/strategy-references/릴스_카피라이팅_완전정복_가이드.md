# 릴스 카피라이팅 완전 정복 가이드

> 콘텐츠 기획 시 참고용 레퍼런스. 릴스 주제 선정·후킹 설계 전 필독.

---

## Chapter 0. 왜 대부분의 릴스는 실패하는가

### 전문가의 함정
- 잠재 고객 100명 중: 이미 관심 있는 사람 5% / 필요성 인지 15% / 무관심 80%
- 전문가는 본능적으로 5%를 위한 콘텐츠를 만든다
- **릴스 성공 구조**: 80% 후킹 → 15% 흥미 유발 → 5% 전문성 증명

### 알고리즘 작동 순서
1. 랜덤 100명에게 노출
2. 첫 3초 이탈률 측정
3. 이탈률 70% 이하 → 1,000명 노출
4. 평균 시청 50% 이상 → 10,000명 노출
5. 저장/공유/댓글 비율 높으면 → 100,000명+ 노출

## Chapter 1. 보편적 흥미 설계 프레임워크

### 관심의 동심원 이론
- 핵심 관심층 (5%): 직접 관련자
- 잠재 관심층 (15%): 필요성 인지자
- 무관심층 (80%): 모르거나 관심 없음

**릴스 접근법**: 무관심층(80%) 먼저 후킹 → 잠재층(15%) 흥미 → 핵심층(5%) 전문성 증명

### 후킹의 3단계 변환 공식

**1단계: 전문 키워드 추출**
- 가르치고 싶은 핵심 개념 정의
- 예: "디버깅", "프롬프트 최적화", "퍼널 설계"

**2단계: 7가지 보편적 흥미 카테고리로 래핑**
| 카테고리 | 핵심 감정 |
|:---|:---|
| 인간관계 갈등 | 공감, 대리만족 |
| 시간/돈 제약 | 불가능의 도전 |
| 신분 전복 | 통쾌함 |
| 몰래카메라 | 호기심 |
| 극단적 도전 | 긴장감 |
| 일상 불편 해결 | 실용성 |
| 권위 파괴 | 통쾌함 |

**3단계: 결과+반전 구조 설계**
- 패턴 1: 감정 반전 (화낼 줄 알았는데 울었다)
- 패턴 2: 결과 반전 (실패할 줄 알았는데 성공)
- 패턴 3: 정체 반전 (가짜일 줄 알았는데 진짜)
- 패턴 4: 과정 반전 (어려울 줄 알았는데 쉬웠다)

### 전문성 노출 3단계
- Level 1: 결과만 노출 (간접)
- Level 2: 과정 노출 (중간)
- Level 3: 해설로 증명 (직접, 신중히)

**최적 조합**: 처음 70% Level 1 → 중간 20% Level 2 → 마지막 10% Level 3

## Chapter 2. 6대 후킹 패턴

### 패턴 1: 시간 압박 + 극단적 결과
**공식**: [시간] + [난이도 높은 결과물] + [실패 시 손해]
- "10분 안에 앱 못 만들면 맥북 포맷합니다"
- 화면에 타이머 표시 필수

### 패턴 2: 관계 기반 드라마
**활용 관계 12가지**: 형제, 부모, 친구, 연인, 동료, 상사, 고객, 경쟁자, 선후배, 이웃, 반려동물, 자기 자신

**구성 4단계**:
1. 문제 상황 소개 (3초)
2. 개입 계획 (2초)
3. 실행 과정 (10초)
4. 반응 포착 (10초)

### 패턴 3: 계급 전복 (저예산 vs 고품질)
**3가지 제약 축**:
- 금액 제약: 만 원 / 5천 원 / 무료
- 도구 제약: 핸드폰 / 무료 앱
- 시간 제약: 10분 / 1시간

### 패턴 4: 지식 전달 재포장
**4가지 권위 설정**:
1. 전문가 빌리기: "구글 개발자가 쓰는..."
2. 계층 설정: "업계 1%만 아는..."
3. 희소성: "처음 공개하는..."
4. 결과 역산: "조회수 100만 만든 방법"

**접근성 보장 키워드**: 집에서 / 쉽게 / 10분 만에 / 무료로 / 초보도 / 혼자서

### 패턴 5: 메타 인지 유도 (고수 vs 초보)
**5가지 비교 축**: 성과 / 효율 / 숙련도 / 결과 / 태도

**스크립트 5단계**:
1. 후킹 질문 (3초)
2. 호기심 증폭 (3초)
3. 차이점 비교 (15초)
4. 구체적 사례 (5초)
5. 자가 진단 + CTA (4초)

### 패턴 6: 권위 숨기기 (전문성 간접 노출)
- 0~10초: 전문성 완전 숨김 (보편적 상황)
- 10~20초: 전문성 힌트 (과정 흘끗)
- 20~28초: 전문성 자연 드러남
- 28~30초: 전문성 명시 (선택적)

## Chapter 3. 주제 설계 실전 프로세스

### 4단계 주제 개발
1. 전문 키워드 추출 (5분)
2. 7가지 보편 흥미 카테고리 매칭 (10분)
3. 상황 드라마 설계 (15분)
4. 후킹 문장 작성 (5분)

### 후킹 문장 공식 5가지
1. [극단 행동] + [대상] + [반응 예고]
2. [시간 제한] + [불가능 목표] + [실패 선언]
3. [저예산] + [고급 결과] + [속이기 실험]
4. [결정적 차이] + [자가 진단]
5. [전문가 방법] + [접근성] + [즉시 적용]

### 주제 검증 5단계 필터
1. 보편성 테스트 (비전공자도 클릭할까?)
2. 드라마 강도 측정 (끝까지 볼까?)
3. 전문성 자연 노출 확인
4. 후킹 문장 강도 테스트 (3초 룰)
5. 실용성 확인 (나도 해볼 수 있겠다?)

## Chapter 4. 플랫폼별 최적화

### 인스타그램 릴스
- 최적 길이: 15~30초
- 자막 필수 (85%가 무음 시청)
- 해시태그: 5~8개
- 저장 가치 강화 우선

### 유튜브 쇼츠
- 최적 길이: 30~60초
- 구독 전환율 중요
- 롱폼 영상 유입 유도

### 틱톡
- 최적 길이: 21~34초 (완주율 최고)
- 트렌드 사운드 필수
- 반복 재생 설계 중요

## Chapter 5. 촬영 & 편집 핵심

### 편집 순서 (캡컷 기준)
1. 컷 편집 (10분) — 2~3초마다 화면 전환
2. 자막 추가 (5분) — AI 자막, 굵은 흰색+검정 테두리
3. 효과 추가 (10분) — Zoom 트랜지션, 속도 조절
4. 음악 삽입 (3분) — 볼륨 20~30%
5. 색보정 (3분) — 밝기+10~20, 채도+5~10
6. 최종 검토 (5분)

### 편집 템플릿 5가지
1. 시간 압박 챌린지
2. 관계 반응 촬영
3. 계급 전복 검증
4. 고수 vs 초보 비교
5. 전문가 방법 재포장

## Chapter 7. 비즈니스 연결

### 4단계 전환 퍼널
1. 인지 (보편적 흥미 릴스) → 조회수
2. 관심 (전문성 자연 노출) → 저장률
3. 고려 (시리즈, 깊이 있는 정보) → 팔로우
4. 전환 (오퍼, 무료 체험) → 매출

### 수익화 모델 10가지
1. 직접 제품/서비스 판매
2. 리드 생성 (이메일 수집)
3. 제휴 마케팅
4. 브랜드 협찬 (1만: 30~50만원/건)
5. 커뮤니티/구독 모델
6. 디지털 제품 판매
7. 오프라인 연계
8. 강의/코칭
9. 컨설팅
10. 플랫폼 광고 수익
