## 핵심 발견: 고성과 캡션의 공통 구조

### 기본 공식 (3단계)
```
[1줄 후킹] — 릴스와 다른 각도로 호기심 또는 공포 유발
↓
[가치 본문] — 자료/팁/정보의 미리보기 또는 근거
↓  
[CTA] — 팔로우 + 댓글 [키워드] → DM 자료 전송
```

## 후킹 유형별 캡션 공식

### 유형 A — "설마 아직도...?" 창피감 유발
```
"설마 아직도 [구식 방법] 하시는 건 아니시죠?"
"아직도 [불편한 것] 쓰고 계신건 아니시죠?"
```
**사용 예시:**
- "설마 아직도 돈 주고 AI 사용하시는거 아니시죠?"
- "설마 아직도 뻔한 AI로 뻔한 상세페이지 만들고 계신건 아니시죠?"
- "아직도 AI로 티나는 글 쓰고 계신건 아니시죠?"

### 유형 B — "이거 안 하면 [나쁜 일]" 공포/손실 유발
```
"gpt 이거 안 하면 개인정보 100% 다 털립니다"
"이거 하기 전에 인스타그램 절대 하지 마세요"
"이 3가지 세팅 안 하고 인스타 올리면 손해입니다"
```
> 손실 회피 심리 극대화. 긴급성 부여.

### 유형 C — 무료 + 희귀 정보 강조
```
"제발 돈 주고 [X] 배우지마세요. [무료 대안] 있습니다."
"이걸 몇 백만 원 받고 알려주는 분들도 계시더라구요? 근데 그냥 제가 무료로 알려드리겠습니다."
"구글이랑 오픈ai에서 직접 무료로 다 알려주는데 도대체 왜 돈 주고 배우고 계신가요"
```

### 유형 D — 커뮤니티/권위 증거
```
"저희 9000명 커뮤니티에선 다들 여기를 사용하고 있습니다"
"약 10,000명 AI 커뮤니티를 운영하는 저도..."
"5년 동안 상세페이지 누적 매출만 15억..."
```

## 고성과 캡션 본문 구조

### 구조 ① @yeongseon.ai 고정 패턴
```
[후킹 1줄]
[핵심 가치 또는 문제 2~3줄]
─────────────────────
여러분은 그냥 제가 보내드리는 자료 그대로 받아가세요.
직접 찾고 공부하고 알게 된 정보 나누어드리겠습니다.
이제 약간의 전략만 알면 같은 시간에도 10배 더 많은 일을 할 수 있는 시대입니다.
남들보다 빠르게 정보를 받아서 실행해보세요.
여러분은 그냥 그대로 따라만 하면 됩니다.
팔로우하고 댓글을 남겨주시면 자료와 링크 공유해드릴게요.
```
> 이 패턴이 모든 게시물에 동일하게 반복됨. 브랜드 일관성 + 신뢰감 형성.

### 구조 ② 인스타 성장 팁 캡션 패턴
```
[놀라운 주장/데이터]
✅/1️⃣ [항목 1]
✅/2️⃣ [항목 2]
...
🔥 결론: [핵심 공식]
[CTA] "댓글에 [키워드] 남기면 자료 바로 드려요"
```

## 검증된 인스타그램 알고리즘 공식 (캡션에서 추출)

### 조회수 요소 점수표
```
해시태그     → 0/10  (조회수랑 상관없음)
후킹         → 8/10  (3초 안에 시선 잡기)
장비         → 0/10  (스마트폰 하나로 충분)
CTA          → 10/10 (저장·공유·팔로우 유도 필수)
상단 카피    → 9/10  (첫 자막이 조회수를 결정)
1일 1포스팅  → 0/10  (무의미, 질이 양보다 중요)
트렌드       → 10/10 (최신 트렌드 = 알고리즘 직행 티켓)

공식: 트렌드 + CTA + 상단카피 + 후킹 = 떡상
```

### 2025 인스타그램 트렌드 변화
```
해시태그 → 캡션 SEO
조회수   → 시청지속시간
릴스     → 릴스 + 게시물
짧은 릴스 → 15-30초 릴스
B-roll  → 스토리텔링
저장     → 좋아요, 공유
캡션유도 → 댓글유도
```

## 후킹 트렌드 5가지 (지금 쓰면 떡상)

| # | 유형 | 설명 | 적합한 콘텐츠 |
|:---:|:---|:---|:---|
| 1 | 복제 후킹 | 두 가지를 나란히 비교 | 비교·정보 콘텐츠 |
| 2 | 매치 컷 후킹 | 같은 장소, 다른 시간대 | 변화·성장 콘텐츠 |
| 3 | POV 후킹 | 1인칭 시점 체험 | 언박싱·리뷰 |
| 4 | 질문형 후킹 | "여러분 그거 아세요?" | 정보·팁 콘텐츠 |
| 5 | 위협·금지 후킹 | "이렇게 하면 절대 안 됩니다" | 경고·주의 콘텐츠 |

## 인스타 처음 시작 시 9단계 (캡션에서 추출)

1. 수익화 카테고리 정하기
2. 프로필 세팅 (Who·What·Why·CTA)
3. 같은 분야 100만 콘텐츠 30개 정리 (Notion 활용)
4. 대본·후킹 분석은 ChatGPT로
5. 수익화 퍼널 설계 (제휴·전자책·VOD·상품)
6. 댓글 자동화 (Social BIZ, Manychat)
7. 촬영→편집 (스마트폰·Capcut)
8. AB테스트로 성과 추적
9. 33법칙으로 콘텐츠 업로드 반복

## 우리 채널 캡션 작성 템플릿

```
[후킹 1줄] — "설마 아직도...?" 또는 숫자 충격형

[핵심 가치 2~3줄]
- 현직 APM 엔지니어로서 실제로 써보고 검증한 것
- 즉시 써먹을 수 있는 구체적 내용

[키워드 줄] — 검색 가능한 용어 자연스럽게 포함
예: "AI 육아 / AI 활용법 / ChatGPT 프롬프트"

[CTA 1개만]
예: "저장해두고 필요할 때 꺼내보세요 💡"
또는 "댓글에 [키워드] 남기면 자료 보내드릴게요"

[해시태그] 3~5개 (니치 특화)
예: #AI활용 #AI육아 #ChatGPT활용법
```

**피해야 할 것:**
- 저장 + 공유 + 댓글 + 팔로우 동시 요청 (아무것도 안 하게 됨)
- "이번 릴스에서는..."으로 캡션 시작
- 해시태그 20개 이상
