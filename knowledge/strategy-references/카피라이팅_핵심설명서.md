# 10배 더 잘 팔리는 카피라이팅 핵심 설명서

> 콘텐츠 제작 시 참고용 레퍼런스. 릴스/캐러셀/캡션 작성 전 필독.

## 핵심 원칙 20개 요약

### 1. 카피 = 심리 설계
- 구조: 공감 → 욕망 → 증거 → 행동
- 무의식 트리거 설계가 핵심

### 2. 5초 안에 의심→확신 (Hook-Heart-Hand)
- Hook: 시선 붙잡기
- Heart: 공감 ("나도 이랬는데")
- Hand: 즉시 행동 유도

### 3. 3대 본능
- 생존 본능: 손실 회피 ("안 쓰면 손해")
- 사회 본능: 인정 욕구 ("상위 0.1%만 아는")
- 편의 본능: 에너지 절약 ("10분이면 끝")

### 4. 4대 심리코드
- 손실회피 (Loss Aversion): 이득보다 손실 2.3배 강하게 느낌
- 희소성 (Scarcity): "오늘까지만", "선착순"
- 권위 (Authority): 숫자+소속+경력
- 사회적 증거 (Social Proof): "2만 명이 이미"

### 5. F.S.P.A 공식
| 단계 | 의미 | 예시 |
|:---|:---|:---|
| F (Fear) | 불안 자극 | "이걸 모르면 이미 손해" |
| S (Solution) | 해결책 | "단 한 문장으로 AI가 대신" |
| P (Proof) | 증거 | "2만 명 검증, 평균 2시간 절약" |
| A (Action) | 행동 유도 | "지금 클릭해서 무료로" |

### 6. 금지형 카피 (역심리)
- "하지 마세요" → 오히려 클릭 유발
- 반드시 대안 제시: "~하지 말고, ~하세요"

### 7. 황금 숫자
- 0.1% / 10분 / 3가지 / 10배
- 희소성·편의성·효율성·권위 동시 전달

### 8. 타겟 명시형 (전환율 3배)
- "모두에게 말하는 문장은 누구에게도 닿지 않는다"
- "직장인이라면", "워킹맘이라면", "왕초보라면"

### 9. 감정 롤러코스터
- 두려움 → 안도 → 성취 순서로 설계

### 10. CTA = 조건부 약속 (If-Now-Then)
- "지금 저장하면(If) → 내일 쓸 때(Now) → 바로 활용(Then)"

### 11. AIDA 변형
- Attention → Interest → Desire → Action
- 시선의 흐름이 아닌 **감정의 순환**

### 12. 30초 전환 구조 5단계
1. 문제 제기: "혹시 이런 경험?"
2. 불안 조성: "그럼 이미 손해"
3. 대안 제시: "이 방법 하나면 끝"
4. 증거 제시: "2만 명 검증"
5. 즉시 행동: "지금 링크 클릭"

### 13. 비교가 최고의 설득
- 10년 VS 1달 / 100시간 VS 10분 / 300만 원 VS 0원

### 14. 팔리는 문장 5종
| 유형 | 구조 |
|:---|:---|
| 공포형 (FOMO) | [금지] + [손실] + [대안] |
| 기회형 (희소) | [상위 0.1%] + [비밀] + [즉시성] |
| 편리형 (효율) | [문제] + [지름길] + [보상] |
| 감정형 (공감) | [일상] + [감정] + [대안] |
| 결과형 (보상) | [숫자] + [효과] + [즉시성] |

### 15. 브랜드형 카피 (장기 신뢰)
- "AI가 일하면, 인간은 창조한다."
- "AI는 사람을 대체하지 않는다. 해방시킨다."

### 16. A/B 테스트 필수 항목
| 항목 | 방식 | 지표 |
|:---|:---|:---|
| 헤드라인 | 감정형 vs 수치형 | CTR |
| CTA | 명령형 vs 약속형 | CVR |

### 17. AI 시대 카피 경쟁력
- AI가 하는 것: 속도, 양, 어조, 문법
- 인간이 해야 하는 것: 맥락 판단, 진심 설계, 심리적 공감

### 18. 카피 5대 금기
1. "모두를 위한" 문장
2. "좋아요·구독" 직접 요청
3. "누구나 할 수 있다" 모호한 약속
4. 검증 없는 수치
5. "도와드리겠습니다" (주체를 고객으로 전환할 것)

### 19. 실행 루틴
1. 첫 문장: 두려움 or 희소성으로 시작
2. 중간: 공감 + 가치
3. 마지막: 즉시 행동 유도
4. 숫자와 증거로 신뢰
5. 24시간 내 A/B 반응 검증

### 20. 핵심 문장
> "클릭은 시선으로 만들고, 매출은 문장으로 만든다."
> "팔리는 문장의 목표는 감동이 아니라 행동이다."
