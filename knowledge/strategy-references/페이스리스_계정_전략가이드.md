## 왜 페이스리스 계정이 빠르게 성장하는가

- **프라이버시 욕구** — 개인 정보 보호하면서 오디언스 구축
- **콘텐츠 우선 문화** — 외모보다 가치에 반응
- **AI 툴 발전** — 카메라 없이 고품질 콘텐츠 가능
- **알고리즘** — 페이스리스 계정이 일반 계정 대비 최대 32% 높은 인게이지먼트

| 장점 | 의미 |

| 프라이버시 | 개인 생활과 공개 활동 분리 |

| 낮은 제작 비용 | 조명·카메라 불필요 |

## 바이럴 릴스 구조 (5요소)

### 1. 훅 — 첫 3초 (가장 중요)

**5가지 훅 패턴:**

| 패턴 | 예시 |
|:---|:---|
| 숫자 폭탄 | "직장인의 95%가 이걸 모른다" |
| 반직관적 주장 | "더 열심히 일하는 것이 당신을 가난하게 만든다" |
| 직접 타깃 지목 | "아직도 ChatGPT를 남들처럼 쓰고 있다면 낭비" |
| 시각적 충격 | 첫 1초에 "이게 뭐지?" 이미지 |
| 오픈 루프 | "모든 걸 해결하는 AI 툴을 찾았다. 어떻게 됐는지 공유한다" |

**물리적 조건:**
- 텍스트는 0.5초 이내에 화면에 있어야 함
- 핵심 텍스트 위치: 상단 또는 중앙 1/3
- 하단 20%는 인스타 UI 버튼에 가려짐 → 절대 핵심 텍스트 금지

### 2. 메인 콘텐츠 — 3~25초

**3가지 검증된 구조:**

**구조 A — 문제 → 해결책**
```
훅: 문제 제시 → 문제 구체화(공감) → 해결책 → 결과 → CTA
```

**구조 B — 카운트다운 리스트**
```
훅: "AI 사용법을 바꿀 3가지" → 항목 3(약) → 항목 2 → 항목 1(강) → CTA
```
> 가장 강한 것을 마지막에 → 완료율 높아짐

**구조 C — 반전 서사**
```
훅: 일반 통념 → "하지만 실제로는..." → 반전 → 이유 → 적용/CTA
```

**핵심 원칙: 릴스 하나에 아이디어 하나**

### 3. 비주얼 스타일

- **컬러 시스템**: 고정 2개 브랜드 컬러 → 모든 콘텐츠에 사용
- **텍스트 애니메이션**: 한 줄씩 나레이션에 맞게 등장
- **컷 속도**: 에너지 높은 콘텐츠 1~2초 / 감성 콘텐츠 3~5초
- **컬러 그레이딩**: 단일 프리셋 고정 사용 (CapCut 저장)

### 4. 캡션 전략

```
1줄: 릴스와 다른 각도 (릴스 내용 반복 금지)
2~4줄: 릴스에서 보여주지 않은 추가 가치
키워드: 검색 가능한 용어 2~3개 자연스럽게
마지막: 댓글 유발하는 구체적 질문 하나
```

### 5. CTA — 릴스당 하나만

| CTA 유형 | 적합 상황 | 예시 |
|:---|:---|:---|
| 저장 유도 | 캐러셀, 팁 릴스 | "스크롤하기 전에 저장하세요 — 나중에 필요해요" |
| 공유 유도 | 명언, 감성 콘텐츠 | "이게 필요한 사람에게 보내주세요" |
| 댓글 유도 | 의견 주제 | "이런 경험 있으면 YES 댓글 달아주세요" |
| 팔로우 유도 | 성장 단계 | "매일 AI 팁 올려요 — 팔로우하면 놓치지 않아요" |

## 초보자 90일 전략

### 1단계: 포스팅 스케줄

| 기간 | 빈도 | 목표 |
|:---|:---:|:---|
| 1~4주 | 주 4회 | 데이터 수집 |
| 5~12주 | 주 5회 | 바이럴 포맷 발견 |
| 3개월 이후 | 주 5~6회 | 복리 성장 |

**일별 배치 (예시):**
```
월: 캐러셀 (저장 목표)
화: 팁 릴스 (도달 목표)
목: 팁 릴스 (도달 목표)
금: 실험적 릴스 (포맷 테스트)
토: 캐러셀 (저장 목표)
```

### 2단계: AI 콘텐츠 아이디어 예시

```
"이 AI 툴 하나가 내 업무 3시간을 4분으로 줄였다"
"아직도 ChatGPT를 이렇게 쓰고 있다면 낭비하는 거다"
"아무도 알려주지 않는 무료 AI 툴 — 이유가 있다"
"ChatGPT vs. Claude vs. Gemini — 실제 비교"
"이메일 쓰는 방식을 바꾼 프롬프트 하나"
```

### 3단계: 30일 추적 지표

| 지표 | 목표 | 의미 |
|:---|:---:|:---|
| 저장률 | 2%+ | 콘텐츠 장기 가치 |
| 공유율 | 1%+ | 입소문 |
| 완료율 | 70%+ | 훅+구조 작동 여부 |
| 팔로우 전환율 | 0.5%+ | 니치 명확성 |

> **30일 후**: 저장률 상위 3개 분석 → 2개월차에 그 포맷 두 배로 늘리기

## 팔로워 10만 달성 계정들의 공통 패턴 7가지

1. **계정 정체성이 한 문장으로 설명된다** — 3초 안에 가치 제안 이해
2. **콘텐츠 포맷이 고정되어 있다** — 주제만 바뀌는 반복 템플릿
3. **모든 포스트가 저장 또는 공유를 위해 설계된다**
4. **비주얼 정체성이 일관된다** — 컬러·폰트 절대 안 바뀜
5. **니치가 좁다** — "AI 프리랜서 생산성"이지 "생산성"이 아님
6. **처음 4~6개월을 버텼다** — 실패 계정의 대부분이 이 시기에 포기
7. **수익화가 처음부터 설계됐다**
