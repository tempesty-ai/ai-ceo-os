# [기획] Claude Design 릴스 제작 기획안

> 작성일: 2026-04-18
> 대상: AI+인사이트 채널 첫 게시물 (인스타그램 릴스)
> 총 길이: 30초 / 1인 제작 기준

---

## 전제 조건

- Claude Design은 현재 **리서치 프리뷰** 상태 (정식 출시 아님)
- 실제 UI 화면이 공개되지 않았을 수 있음 → 공식 트위터/X, Anthropic 홈페이지 캡처로 대체
- 1인 제작: 스마트폰 + 삼각대 + 무료 편집앱(CapCut 기준)

---

## 1. 씬별 촬영 구성

### Scene 1 — 0~3초 (후킹)
- **화면 구성**: 얼굴 등장. 정면 상반신 프레임 (어깨~머리). 배경은 노트북 화면이 흐릿하게 보이는 책상.
- **촬영 방법**: 스마트폰 고정(삼각대), 눈높이 카메라. 자연광 또는 링라이트 1개.
- **행동**: 카메라 정면 응시하며 대사 전달. 손동작 없음 — 얼굴 집중.
- **목적**: 첫 프레임에 사람 얼굴 → 인스타 알고리즘 이탈 방지.

### Scene 2 — 4~10초 (제품 설명)
- **화면 구성**: 얼굴 → 화면 녹화 전환 (스크린 레코딩 or 이미지 슬라이드).
- **소스**: Anthropic 공식 발표 이미지, 트위터/X 스크린샷, 유튜브 데모 클립(저작권 확인 후).
- **편집 처리**: 화면 위에 자막 오버레이로 "와이어프레임", "프로토타입", "슬라이드덱" 키워드 타이핑 효과.
- **대안 (소스 없을 경우)**: 화이트보드 앱(Miro 등)에서 직접 "와이어프레임 → 프로토타입 → 슬라이드덱" 플로우 도식 제작 후 화면 캡처.

### Scene 3 — 11~20초 (차별화 포인트)
- **화면 구성**: 얼굴 복귀 (정면). 또는 얼굴 + 우측에 텍스트 카드 분할 화면.
- **촬영 방법**: Scene 1과 동일 세팅. 연속 촬영해서 후편집으로 컷.
- **텍스트 카드 삽입**: "하나의 대화창에서 끝" → "Claude Code로 바로 넘김" 두 줄 카드.
- **행동**: "하나의 대화창 안에서"에서 손으로 원을 그리는 제스처 1회.

### Scene 4 — 21~27초 (임팩트 — 업계 파장)
- **화면 구성**: 화면 녹화. 구글에서 "Paper app", "Pencil app" 검색 화면 실시간 캡처.
- **편집 처리**: 해당 앱 로고 옆에 빨간 "위기" 이모지 or 텍스트 "비상" 삽입. 0.5초 flash cut.
- **이후**: 얼굴 컷으로 복귀. "현직 엔지니어 입장에서" 발화 시 화면 응시.
- **대안 (빠른 제작용)**: 얼굴 컷 유지하고 화면 하단에 "Paper / Pencil → 직격탄" 자막으로 대체.

### Scene 5 — 28~30초 (CTA)
- **화면 구성**: 얼굴 정면. 살짝 가까이 줌인 (편집으로 처리, 재촬영 불필요).
- **행동**: "팔로우해두세요"에서 카메라 방향으로 손가락 한 번 가리키기.
- **화면 하단**: "팔로우" 텍스트 + 화살표 애니메이션 (CapCut 기본 템플릿).

## 2. 필요한 소스 영상/이미지 리스트

| 번호 | 소스 | 확보 방법 | 우선순위 |
|:--|:--|:--|:--|
| 1 | Claude Design 공식 발표 이미지 | Anthropic 트위터/X(@AnthropicAI) 스크린샷 | 필수 |
| 2 | Claude Design 데모 영상 클립 | 유튜브 검색 "Claude Design demo" → 스크린 레코딩 (퍼블릭 데모에 한함) | 선택 |
| 3 | Anthropic 공식 홈페이지 발표 페이지 | 브라우저 캡처 | 필수 |
| 4 | Paper 앱 / Pencil 앱 검색 화면 | 구글 검색 후 스크린 레코딩 | 선택 |
| 5 | "와이어프레임→코드" 플로우 도식 | 직접 Miro or Figma Slides로 5분 제작 | 소스 없을 때 대체 |

> 주의: 타사 앱 로고 사용 시 비방 의도 없이 "업계 파장" 맥락으로만 사용. 로고 단독 클로즈업은 피할 것.

## 3. 편집 구성

### 자막 스타일
- **폰트**: 고딕 계열 굵은 폰트 (CapCut 기본 "굵게" 스타일 또는 Noto Sans Bold)
- **위치**: 화면 하단 1/3 고정
- **색상**: 흰색 텍스트 + 검정 테두리 (가독성 최우선)
- **크기**: 화면 너비 80% 이내, 한 줄에 15자 이내
- **키워드 강조**: "게임 체인저", "하나의 대화창", "Claude Code" → 노란색 또는 강조 컬러 처리

### 트랜지션
- Scene 1 → 2: 컷 편집 (빠른 전환, 효과 없음)
- Scene 2 → 3: 컷 편집
- Scene 3 → 4: 줌아웃 or 슬라이드 트랜지션 (CapCut 기본) 1개만
- Scene 4 → 5: 컷 편집

> 트랜지션 남용 금지. 전환 효과는 전체 1~2개만.

### 텍스트 효과
- Scene 2: "와이어프레임 → 프로토타입 → 슬라이드덱" 타이핑 효과 (CapCut 텍스트 애니메이션 "타이핑")
- Scene 5: "팔로우" 텍스트 + 위아래 바운스 애니메이션

### 배경음악
- 템포 빠른 로파이 비트 or 미니멀 테크 BGM
- 볼륨: 목소리의 20~25% (목소리 우선)
- 소스: CapCut 무료 음원 라이브러리 ("tech", "upbeat" 태그)

### 화면 비율
- 9:16 세로 (인스타 릴스 표준)
- 촬영 시 세로 모드 고정

## 4. 업로드 전 체크리스트

### 콘텐츠 품질
- [ ] 첫 3초 내 얼굴 또는 강한 텍스트 훅 확인
- [ ] 전체 길이 28~32초 범위 내 확인
- [ ] 자막 전 구간 100% 삽입 확인 (소리 꺼도 내용 전달 가능)
- [ ] "리서치 프리뷰" 언급 대사 포함 확인 (과장 방지)
- [ ] CTA 1개만 — "팔로우해두세요" 단일 확인

### 기술 품질
- [ ] 해상도: 1080x1920 이상
- [ ] 오디오: 잡음 없음, 목소리 선명
- [ ] 밝기: 얼굴 어둡지 않음 (링라이트 or 창가 자연광 확인)

### 업로드 세팅
- [ ] 캡션 작성 완료 (별도 작성 필요)
- [ ] 커버 이미지: Scene 1 얼굴 프레임 or "디자이너 없이 와이어프레임부터 코드까지" 텍스트 커버
- [ ] 해시태그: #AI활용 #테크인사이트 #Claude #인공지능 #개발자 (5개 이내 추천)
- [ ] 위치 태그: 선택 (첫 게시물은 생략 가능)
- [ ] 업로드 시간: 평일 오전 7~9시 or 저녁 8~10시 (인스타 피크 타임)

## 촬영 순서 (오늘 바로 실행)

```
1. 소스 수집 (15분)
   → Anthropic 트위터/X에서 Claude Design 발표 이미지 저장
   → Anthropic 공식 사이트 캡처

2. 얼굴 촬영 (20분)
   → 책상 세팅 (노트북 배경), 조명 확인
   → Scene 1, 3, 5 한 번에 연속 촬영 (3~4번 테이크)

3. 화면 레코딩 (10분)
   → Scene 2용 Claude Design 이미지 슬라이드
   → Scene 4용 구글 검색 화면 레코딩

4. CapCut 편집 (40분)
   → 클립 조립 → 자막 → 텍스트 효과 → BGM → 최종 확인

5. 업로드 (5분)
   → 커버 이미지 선택 → 캡션 붙여넣기 → 해시태그 → 게시
```

총 예상 소요 시간: **1시간 30분**
