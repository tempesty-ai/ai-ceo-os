## 아이디어 1 — AI+인사이트

**제목**: `ChatGPT, 이렇게 쓰면 손해`
**형식**: 릴스
**핵심 메시지**: 대부분의 사람이 ChatGPT를 "검색창"으로 쓰는데, 프롬프트 구조 하나만 바꾸면 결과가 10배 달라진다. 비교 시연으로 즉시 체감 가능.
**타겟 단계**: 1단계 (노출 후킹 — 무관심층 80% 타겟)

**후킹 설계**:
- 패턴: 계급 전복 (패턴 3) + FOMO형 카피 (손실회피)
- 후킹 문장: "ChatGPT 이렇게 쓰면 손해입니다" (금지형 카피 → 역심리 클릭 유도)
- 전문성 노출: Level 1 결과 먼저 (비교 화면) → Level 2 과정 (프롬프트 구조 설명)

**제작 메모**:
- 비포/애프터 화면 분할 필수
- 자막 필수 (무음 시청 85%)
- 길이: 20~25초 (완주율 확보)

## 아이디어 2 — AI+인사이트

**제목**: `AI 고수 vs 초보 차이점`
**형식**: 릴스
**핵심 메시지**: 같은 AI 툴을 써도 결과가 극단적으로 갈리는 이유 — "질문 설계" 능력 차이. 고수의 프롬프트 패턴 3가지 공개.
**타겟 단계**: 2단계 (프로필 탐색 — 신뢰 형성)

**후킹 설계**:
- 패턴: 메타인지 유도 (패턴 5) + 기회형 카피 (희소성 + 상위 0.1%)
- 후킹 문장: "AI 고수와 초보, 딱 한 가지가 다릅니다" (메타인지 자가 진단 유도)
- 전문성 노출: 현직 엔지니어 시점 → 권위 설정

**제작 메모**:
- 5단계 구조: 후킹 질문(3초) → 호기심 증폭(3초) → 차이점 비교(12초) → 사례(5초) → CTA(3초)
- CTA: 저장 유도 ("이거 저장해두면 내일 바로 씁니다")

## 아이디어 3 — AI+인사이트

**제목**: `10분에 보고서 끝내기`
**형식**: 캐러셀
**핵심 메시지**: 기획서·보고서 작성에 평균 2~3시간 쓰는 직장인이 AI 프롬프트 3개로 10분 안에 초안 완성하는 실전 템플릿 공개.
**타겟 단계**: 3단계 (팔로우 + 저장 유도 — 즉시 가치 전달)

**후킹 설계**:
- 패턴: 지식 전달 재포장 (패턴 4) + 편리형 카피 (문제 + 지름길 + 보상)
- 후킹 문장: "보고서 2시간? 저는 10분씁니다" (시간 비교 — 황금 숫자 원칙)
- 전문성 노출: 실제 프롬프트 텍스트 노출 (저장 가치 극대화)

**제작 메모**:
- 캐러셀 구성: 1장(후킹) → 2~4장(프롬프트 3개 각 1장) → 5장(CTA: 저장 유도)
- 저장율이 가장 높은 포맷 — 실전 템플릿 형태로 제작
- 총 5장 구성 권장

## 아이디어 4 — AI+육아

**제목**: `육아 고민, AI한테 물어봤다`
**형식**: 릴스
**핵심 메시지**: 아이 거절 기술, 수면 루틴 설계 등 육아 고민을 AI에게 구체적으로 질문하는 방법. "AI가 무슨 육아를 알아?" → 실제 답변 시연으로 반전.
**타겟 단계**: 1단계 (노출 후킹 — 육아 관심층 진입)

**후킹 설계**:
- 패턴: 결과 반전 (패턴 4 과정 반전) + 감정형 카피 (일상 + 감정 + 대안)
- 후킹 문장: "육아 고민, AI한테 물어보니 소아과 선생님 수준 나왔습니다" (정체 반전)
- 전문성 노출: 엔지니어가 설계한 프롬프트 구조 자연 노출

**제작 메모**:
- 반드시 "전문가 권고 없는 건강/발달 조언 금지" 원칙 준수
- 의학 정보 아닌 "대화·루틴·놀이" 영역으로 제한
- "참고용입니다, 전문가 상담 우선" 자막 1회 삽입 필수

## 아이디어 5 — AI+인사이트 (교차: 직장인+부모)

**제목**: `하루 2시간 아끼는 법`
**형식**: 캐러셀
**핵심 메시지**: 현직 엔지니어가 실제로 쓰는 AI 자동화 루틴 5가지. 업무+육아 병행하는 워킹맘·워킹대디 타겟. "나도 바빠서 못 한다"는 반박을 사전 차단.
**타겟 단계**: 2~3단계 (신뢰 형성 → 팔로우 전환)

**후킹 설계**:
- 패턴: 시간 압박 + 결과형 카피 (숫자 + 효과 + 즉시성)
- 후킹 문장: "하루 2시간 아낍니다 — 현직 엔지니어 루틴 5가지" (결과 역산 + 권위)
- 전문성 노출: "현직 엔지니어"라는 권위를 자연스럽게 — Level 2 수준

**제작 메모**:
- 캐러셀 구성: 1장(후킹) → 2~6장(루틴 5개 각 1장) → 7장(CTA: 팔로우 유도)
- 타겟 명시: "워킹맘·워킹대디라면" 첫 장에 삽입 (전환율 3배 원칙)
- 저장 + 공유 동시 유도 가능한 포맷
