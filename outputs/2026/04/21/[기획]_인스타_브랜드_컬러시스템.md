## 핵심 컨셉

**"차갑지 않은 테크"** — 따뜻하고 지적이며, 바로 써먹을 수 있다는 신뢰감

대부분의 AI 콘텐츠 계정이 차가운 파란색 계열(미래적, 기계적 느낌)을 쓸 때,
우리는 따뜻한 파치먼트 + 테라코타 계열로 차별화한다.

## 슬라이드 유형별 적용 가이드

### TYPE A — 라이트 슬라이드 (기본)
```
배경: #F5F4ED (파치먼트)
제목: #141413 (딥블랙), 굵게
본문: #5E5D59 (올리브그레이)
포인트 단어: #C96442 (테라코타)
구분선: #E8E6DC
```
> 사용: 캐러셀 일반 슬라이드, 정보 전달 컷

### TYPE B — 다크 슬라이드 (임팩트)
```
배경: #141413 (딥다크)
제목: #FAF9F5 (아이보리)
본문: #B0AEA5 (웜실버)
포인트 단어: #D97757 (코랄)
구분선: #30302E
```
> 사용: 첫 번째 슬라이드 훅, 숫자 강조 컷, 마지막 CTA 컷

### TYPE C — 인용/팁 박스
```
배경: #FAF9F5 (아이보리)
테두리: 1px solid #E8E6DC
텍스트: #141413
강조: #C96442
```
> 사용: "핵심 한 줄" 박스, 꿀팁 강조 구역

## 타이포그래피 방향

### 원칙
- **한국어 세리프 계열** → 헤드라인에 권위감 (예: Noto Serif KR, 나눔명조)
- **한국어 산세리프 계열** → 본문/UI에 가독성 (예: Noto Sans KR, 프리텐다드)
- 폰트 굵기: 헤드라인은 Medium(500) 유지 — Bold 남발 금지

### Canva 추천 폰트 조합
| 역할 | 추천 폰트 |
|:---|:---|
| 헤드라인 | 나눔명조 / Noto Serif KR Medium |
| 본문 | 프리텐다드 Regular / Noto Sans KR |
| 강조 숫자 | 프리텐다드 SemiBold |

## 릴스 자막 / 썸네일 룰

### 자막
- 배경: `#F5F4ED` 반투명 박스 (opacity 85~90%)
- 텍스트: `#141413`
- 강조 단어: `#C96442`
- 테두리: `1px solid #E8E6DC`

### 썸네일 (첫 프레임)
- 배경: 딥다크(`#141413`) 또는 파치먼트(`#F5F4ED`) 중 택일 — 매 콘텐츠 교차 사용
- 주제어: 크고 굵게, 테라코타 포인트 1단어
- 얼굴 노출 첫 프레임 정책 유지

---

## 피드 통일성 규칙

1. **교차 배치**: 라이트 슬라이드 ↔ 다크 슬라이드 번갈아 게시 (피드 리듬)
2. **포인트 컬러 1개만**: 테라코타(`#C96442`) 하나만 — 다른 색 추가 금지
3. **배경 2종 고정**: 파치먼트(`#F5F4ED`) / 딥다크(`#141413`) — 그 외 배경 금지
4. **텍스트 계층 3단계**: 제목(크고 어둡게) → 본문(중간) → 보조(작고 연하게)
