## 슬라이드 6 — 진짜 꿀팁 (가치 전달)

**헤더:** `실제로 써먹는 Claude Code 꿀팁`

**본문:**

```
1. 파이프 활용
cat 에러로그.txt | claude -p "이 에러 원인은?"
→ 파일 내용 바로 분석

2. 비인터랙티브 모드
claude -p "이 코드 리뷰해줘" --output-format json
→ 자동화 파이프라인에 붙이기 가능

3. 프로젝트 지침
CLAUDE.md 파일에 규칙 써두면
→ 모든 대화에 자동 적용

4. 대화 이어받기
claude --continue
→ 직전 세션 그대로 재개
```
