# 프로필 소개글 3버전
> 기준: @newsdaddy.ai 분석 + [직업] + [팔로워가 얻는 것] + [행동 유도] 3줄 구조
> Instagram 소개글 제한: 150자

---

## 버전 1 — 전문가 포지셔닝 (신뢰 먼저)

```
APM 품질 엔지니어가 직접 써보는 AI 육아 🤖
딸 키우며 찾은 진짜 쓸만한 AI 꿀팁 매주 공유
👇 이번 주 프롬프트 무료 받기
```

**타겟**: AI에 관심 있는 부모, 기술 신뢰도 중시
**강점**: "현직 엔지니어"가 필터링했다는 신뢰감
**약점**: 딸과의 감성 연결이 약함

---

## 버전 2 — 딸+감성 포지셔닝 (공감 먼저) ★ 추천

```
딸이랑 AI 실험 중인 엔지니어 아빠 🧪
"이게 진짜 되네?" 매주 기록합니다
💬 댓글에 "프롬프트" → 바로 DM드려요
```

**타겟**: 육아 중인 부모, 밈 팔로워 → 일반 팬
**강점**: 바이럴 영상(캐치캐치)의 감성과 연결 / 댓글 유도 CTA 포함
**약점**: 전문성 표현이 약간 약함

**추천 이유**: 현재 팔로워 구성(밈 유입)과 연속성 있음. "딸이랑" 키워드가 캐치캐치 밈과 연결됨.

---

## 버전 3 — 실용 포지셔닝 (가치 먼저)

```
AI 몰라도 괜찮아요, 딸도 씁니다 🐾
현직 엔지니어의 AI 육아 실전 로그
👇 프롬프트 받아가세요 (무료)
```

**타겟**: AI 초보 부모, 진입장벽 낮추고 싶은 분들
**강점**: "AI 몰라도 된다" → 가장 넓은 타겟
**약점**: 전문가 포지셔닝이 희석될 수 있음

---

## 공통 권장사항

- 프로필 이름: **AI 언더독** (현재 유지) — 바이럴 영상으로 인식된 계정명
- 링크인바이오: 오픈카카오 또는 뉴스레터 중 CEO 결정 필요

  - 🧪 `딸이랑 실험` — 아이+AI 체험 영상
  - 💬 `프롬프트 모음` — 저장용 자료
