### @newsdaddy.ai — **팔로워 74,000명** (스크린샷 확인)

| 항목 | 내용 |
|:---|:---|
| 팔로워 | **7만 4천** |
| 게시물 | 83개 |
| 포지셔닝 | "아빠 + AI 에이전트 실전 교육" |
| 반복 콘텐츠 | "9살도 해내는 AI에이전트 셋업" — 아이+AI 조합 일관 |
| 프로필 사진 | 정장 입은 남성 (전문가 신뢰감) |

**가져갈 것**
- "9살도 할 수 있다" = 난이도 낮춤 + 아이 각도 → 우리 "딸과 함께" 포맷과 동일 원리
- 타이틀 공식: `[낮은 진입장벽] + [AI 기능명] + [완료/결과]`
- 콘텐츠 일관성: 모든 게시물이 "아이+AI에이전트" 하나의 주제로 수렴

**우리와 차이점**
- @newsdaddy.ai는 AI 에이전트에 집중 → 우리는 AI 육아 전반으로 더 넓음
- 우리는 "현직 엔지니어" 전문성이 추가 무기

- 비주얼 위주, 육아 맥락 없음 → 직접 모방 부적합
- 그러나 "PROMPT 공유" CTA 포맷은 차용 가능

## 4. @ai_underdog 즉각 액션 아이템

### 긴급 (이번 주)

1. **프로필 소개글 개편**
   - 현재: 짧고 CTA 없음
   - 목표: `"[직업] + [독자가 얻는 것] + [행동 유도]"` 3줄 구조
   - 참고: @newsdaddy.ai 소개글 포맷

2. **콘텐츠 정체성 통일**
   - 문제: 뉴스 분석 / 도구 소개 / Claude 홍보 혼재
   - 해결: **"현직 엔지니어 아빠가 딸과 써보는 AI"** 각도로 모든 게시물 수렴
   - 모든 썸네일에 일관된 텍스트 오버레이 스타일 적용

3. **게시물 수 확대**
   - 7개 → 최소 20개로 (콘텐츠 신뢰도/프로필 완성도)
   - 레퍼런스 계정 최소 게시물: @newsdaddy.ai 83개로 74k 팔로워

### 중기 (2주 내)

4. **후킹 공식 2개 우선 적용**
   - `"9살도 하는 AI" 패턴` → "5살도 따라하는 AI [기능명]"
   - `"[N년] 쓰고서야 알게 된"` → "엔지니어가 [N]개월 써보고 알게 된"

5. **"PROMPT 공유" CTA 통일**
   - @4reels.pro + @prompts.ig 공통 패턴
   - "댓글에 [키워드] 남기면 프롬프트 DM드려요" → 댓글 인게이지먼트 상승
