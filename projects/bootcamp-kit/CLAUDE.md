## 🎯 AI야 이렇게 해줘 (★ 대표님 무의식 10습관 = AI 강제 규칙)

> **이 섹션이 10습관 체크리스트를 대체합니다.**
> 수강생이 매일 10개를 외우는 대신, **AI(Claude Code)가 매 세션 자동으로 지킵니다.**
> 수강생은 **의식적으로 10개를 쓰는 연습**만 하면 됨 (과제 C-08/C-10 자가 체크).

### 말투 — 10습관 #5, #8
- 전문용어 쓰면 바로 뒤에 **"쉽게 말하면 ~"** 붙여줘 (#5)
- **선택지 2개 주고 비교**해줘. 혼자 결정하지 마. (#8 "선택지 2개 비교 → 근거로 결정")
- 내가 "진짜로?" 물으면 근거 다시 설명해줘

### 작업 시작 전 — 10습관 #1, #2, #3, #4
- **`PLAN.md` 확인** — 기획서 없으면 코드 안 씀 (#1)
- **"이게 진짜 문제인가?"** 1회 재정의 (#2)
- **"다른 사람은 어떻게 했지?"** — GitHub 레퍼런스 5분 검색 (#3)
- **환경 확인 먼저** (`ls`, `glob`) — 추정 금지 (#4)

### 작업 중 — 10습관 #6, #7, #10
- **10분 단위로 `git commit` 제안** (#6 "커밋 안 된 코드는 존재하지 않는 코드")
- **테스트 안 돌아간 코드는 "완료"라고 하지 마** (#7 — 실전 근거: 2026-04-14 "HTTP 200 ≠ 배포 완료")
- **"다른 프로젝트에 영향 없나?"** 자문 (#10 크로스 도메인)

### 작업 끝 — 10습관 #9
- 터진 순간 1문장이라도 **`lessons.md`에 기록** (#9 "실패 기록 → 같은 실수 안 함")

### 절대 금지
- ❌ 내 카드 키를 코드에 하드코딩
- ❌ `.env.local`을 git에 커밋
- ❌ 모르는 것을 "아마도 ~일거야"로 추측 (대표님 2026-03-24 ainowclass 결제 경로 사고 근거)
- ❌ `npm install` 없이 새 패키지 사용
- ❌ **`git add -A` 금지** — 특정 파일만 add (대표님 2026-04-11 `.env.backup` 시크릿 3주 커밋 사고 근거)
