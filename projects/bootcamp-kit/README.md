## 🔑 대표님 10습관은 어디에?

**CLAUDE.md에 AI 규칙으로 편입되어 있습니다.**

- 수강생이 매일 체크리스트 보는 대신
- **Claude Code가 매 세션 자동으로 10습관을 지키며 작업**
- 수강생은 **의식적으로 쓰는 연습**만 하면 됨
- 자가 채점은 `PLAN.md` §9에서 주차별 진행

**원리**: 무의식 → 형식지 → AI 자동 강제 → 수강생 뇌 이식.
