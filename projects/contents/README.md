# 콘텐츠 사업부

> **미션**: 조회수 잘 나오는 콘텐츠, 인기 있는 콘텐츠를 지속적으로 생산하는 시스템.
> **담당**: CCO (주) + CDO (부) + CMO (+1)
> **범위**: 멀티플랫폼 콘텐츠 기획·생산·성과 분석

---

## 구조

```
contents/
├── CLAUDE.md                       ← 사업부 헌법
├── agents/
│   ├── README.md                   ← 임원 선언 (본사 @참조)
│   ├── cco.md                      ← CCO 콘텐츠 특화
│   ├── cdo.md                      ← CDO 콘텐츠 특화
│   └── cmo.md                      ← CMO 콘텐츠 특화
├── context/
│   ├── platform-rules.md           ← 플랫폼별 운영 규칙
│   └── viral-formula.md            ← 바이럴 공식 레퍼런스
├── rules/
│   ├── 금지행동.md
│   └── 품질기준.md
└── skills/
    ├── README.md
    ├── 바이럴-기획.md               ← 핵심 스킬
    ├── 유튜브-기획.md
    ├── 콘텐츠-성과예측.md
    └── 멀티플랫폼-캘린더.md
```

---

## marketing 사업부와 차이

| 구분 | marketing | contents |
|:---|:---|:---|
| 목표 | 팔로워 전환 (여정 1~3단계) | 조회수·바이럴·인기 |
| 주 지표 | 팔로우 전환율, DM 문의 | 조회수, 시청 완료율, 저장·공유율 |
| 플랫폼 | 인스타 중심 | 멀티플랫폼 (인스타+유튜브+틱톡) |
| 핵심 임원 | CCO + CMO + CDO + CXO | CCO + CDO + CMO |
