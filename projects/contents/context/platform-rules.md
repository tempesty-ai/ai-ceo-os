# 멀티플랫폼 운영 규칙

> 콘텐츠 사업부가 운영하는 3개 플랫폼의 알고리즘·포맷·운영 규칙.
> CMO/CCO/CDO 작업 시 반드시 참조.

---

## 인스타그램 릴스

### 알고리즘 신호 우선순위
1. **시청 완료율** (가장 강한 신호) — 30초 릴스 기준 70% 이상 목표
2. **저장률** — 저장이 많을수록 추천 알고리즘 가산점
3. **공유율** — DM 전송·스토리 공유 강력 신호
4. **댓글** — 열린 질문 CTA로 유도
5. **좋아요** — 상대적으로 약한 신호

### 포맷 규칙
- 세로 비율: **9:16** (1080×1920px)
- 길이: **15~60초** 권장 (30~45초 최적)
- 첫 프레임에 자막 표시 (소리 OFF 시청자 공략)
- 트렌드 음악 사용 시 도달 부스트 효과

### 업로드 타이밍
- 권장 시간: **화·목·금 오후 7~9시**, **토·일 오전 10~12시**
- 주 3~5회 일관성 유지 (간격보다 꾸준함이 중요)

### 해시태그 전략
- 총 5~10개 (과도한 해시태그 역효과)
- 구성: 대형(팔로워 100만+) 1개 + 중형(10만~100만) 2~3개 + 소형(1만~10만) 2~3개 + 니치(1만 이하) 1~2개
- 캡션보다 댓글에 해시태그 넣는 실험 병행

### 캡션 규칙
- 첫 2줄이 핵심 (더보기 클릭 전 보이는 영역)
- CTA 1개만 (저장 / 팔로우 / 댓글 중 택1)
- 이모지는 시각적 강조용으로 절제 (문장마다 사용 금지)

## 유튜브 숏츠

### 알고리즘 신호 우선순위
1. **CTR (썸네일 클릭률)** — 숏츠 피드에서 노출 → 클릭으로 이어지는 비율
2. **시청 완료율** — 루프 재생 횟수 포함
3. **댓글 참여** — 질문·토론 유발 CTA 효과적
4. **좋아요·싫어요 비율**

### 포맷 규칙
- 세로 비율: **9:16** (인스타와 동일 원본 활용 가능)
- 길이: **60초 이하** (숏츠 조건)
- 제목에 키워드 포함 필수 (검색 노출)
- 썸네일: 자동 생성 or 커스텀 (커스텀이 CTR 향상)

### 업로드 타이밍
- 매일 1개 이상이 이상적 (숏츠는 양이 질만큼 중요)
- 권장 시간: **오전 9~11시**, **오후 6~8시**

### 캡션/설명란
- 첫 2~3줄에 핵심 키워드 포함 (SEO)
- 구독 CTA + 다음 영상 연결 링크

### 인스타와 차이점
| 인스타 릴스 | 유튜브 숏츠 |
|:---|:---|
| 해시태그 도달이 강함 | 키워드 검색 도달이 강함 |
| 저장률이 핵심 지표 | CTR이 핵심 지표 |
| 음악 트렌드 중요 | 썸네일·제목 중요 |
| 팔로워 피드 중심 | 추천 알고리즘 중심 |

## 틱톡

### 알고리즘 신호 우선순위
1. **시청 완료율 + 루프 재생** (가장 강한 신호)
2. **공유** (바이럴 확산의 핵심)
3. **댓글** (커뮤니티 참여 신호)
4. **좋아요**

### 포맷 규칙
- 세로 비율: **9:16**
- 길이: **15~60초** 최적 (3분까지 가능하나 숏폼 우선)
- 트렌드 사운드 사용 시 For You 피드 노출 부스트
- 텍스트 오버레이 필수 (틱톡도 무음 시청 많음)

### 업로드 타이밍
- 권장: **화·목 저녁 7~9시**, **주말 오후 2~5시**
- 틱톡은 발행 후 72시간이 바이럴 골든타임

### 해시태그
- 3~5개로 간소화 (틱톡은 너무 많으면 역효과)
- `#foryou` `#fyp` + 주제 특화 2~3개

### 인스타·유튜브와 차이점
| 구분 | 틱톡 |
|:---|:---|
| 팔로워 의존도 | 낮음 (비팔로워에게도 강하게 노출) |
| 진입 장벽 | 낮음 (신규 계정도 바이럴 가능) |
| 트렌드 속도 | 빠름 (48~72시간 이내 올라타야) |
| 콘텐츠 톤 | 자연스럽고 날것의 느낌 선호 |

## 멀티플랫폼 크로스포스팅 가이드

### 원칙
**같은 영상 그대로 올리지 않는다.** 각 플랫폼 알고리즘에 맞게 커스터마이징.

| 요소 | 인스타 | 유튜브 숏츠 | 틱톡 |
|:---|:---|:---|:---|
| 캡션 | 2줄 핵심 + CTA | 키워드 포함 설명란 | 짧고 직관적 |
| 해시태그 | 5~10개 | 설명란에 키워드 | 3~5개 |
| 음악 | 인스타 트렌드 음악 | 중립적 BGM | 틱톡 트렌드 사운드 |
| 워터마크 | 인스타 워터마크 제거 후 타 플랫폼 업로드 권장 |

### 크로스포스팅 순서
1. 인스타 릴스 원본 제작·업로드
2. 워터마크 제거 버전으로 유튜브 숏츠 최적화 업로드
3. 틱톡용 트렌드 사운드 교체 후 업로드
