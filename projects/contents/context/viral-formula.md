# 바이럴 공식 레퍼런스

> 검증된 바이럴 패턴과 재현 공식 라이브러리.
> CCO가 콘텐츠 기획 시, CDO가 성과 예측 시 참조.
> 실제 성과 데이터가 쌓이면 CDO가 이 파일을 업데이트한다.

---

## 바이럴의 5가지 메커니즘

콘텐츠가 퍼지는 이유는 5가지 심리 메커니즘 중 하나 이상을 자극했기 때문이다.

| # | 메커니즘 | 핵심 감정 | 콘텐츠 패턴 |
|:---:|:---|:---|:---|
| 1 | **사회적 화폐** | "이걸 알면 나 스마트해 보여" | 잘 알려지지 않은 꿀팁·내부 정보 |
| 2 | **공감 고통** | "나만 그런 게 아니었어!" | 공통 고민·불편함의 정확한 언어화 |
| 3 | **실용 가치** | "이거 바로 써먹어야겠다" | 즉시 활용 가능한 정보·도구 |
| 4 | **스토리 텐션** | "어떻게 됐지?" | 반전·예상 외 결말·성공/실패 서사 |
| 5 | **집단 정체성** | "이건 우리 얘기야" | 특정 집단의 경험·언어·문화 공유 |

> **적용 원칙**: 하나의 콘텐츠에 2개 이상 조합 시 바이럴 확률 상승.

## 후킹 12타입 (검증된 인트로 공식)

| # | 타입 | 템플릿 | 최적 플랫폼 |
|:---:|:---|:---|:---|
| 1 | **질문형** | "근데 왜 아직도 ___하고 있어요?" | 전 플랫폼 |
| 2 | **충격 통계** | "지금부터 ___하는 사람이 N%밖에 없대요" | 인스타·유튜브 |
| 3 | **반전** | "다들 ___가 좋다는데, 사실 반대예요" | 전 플랫폼 |
| 4 | **공감 고통** | "___할 때마다 속 터지는 거 저만 그래요?" | 틱톡·인스타 |
| 5 | **결과 먼저** | "이거 하나로 한 달에 ___ 됐어요" | 전 플랫폼 |
| 6 | **비밀 공개** | "아무도 안 알려주는 ___의 진짜 이유" | 전 플랫폼 |
| 7 | **부정 어법** | "___만은 절대 하지 마세요" | 전 플랫폼 |
| 8 | **권위 인용** | "[유명인/기관]이 ___라고 했는데" | 유튜브·인스타 |
| 9 | **시간 제한** | "딱 30초만 보세요. ___ 알려드릴게요" | 전 플랫폼 |
| 10 | **리스트** | "___하는 3가지 방법" (뻔하면 감점) | 유튜브·인스타 |
| 11 | **자기 반성** | "저도 예전엔 ___ 했거든요" | 틱톡·인스타 |
| 12 | **비교** | "A 하는 사람 vs B 하는 사람" | 전 플랫폼 |

**조합 원칙**: 최소 3개 이상 조합. 타입 1~6이 타입 7~12보다 평균 조회수 높음 (일반적 경향).

## 콘텐츠 구조 공식

### 공식 1: 역순 대본법 (Backward Script)
바이럴 가능성이 가장 높은 구조.

```
Step 1: 최종 감정/행동 1줄 정의
  ↓
Step 2: 핵심 메시지 1줄
  ↓
Step 3: 전환 스토리 (2~3문장)
  ↓
Step 4: 후킹 3개 이상 설계 (첫 3초)
  ↓
Step 5: CTA 1개
```

**사용 시점**: 교육형, 인사이트형, 스토리형 콘텐츠

---

### 공식 2: 문제-공감-해결 (PAS)
저장률·신뢰 자산 콘텐츠에 최적.

```
문제 제시 (공감 유발)
→ 공감 증폭 (얼마나 심각한가)
→ 해결책 제시 (바로 써먹는 것)
→ CTA
```

**사용 시점**: 꿀팁형, 저장 유도형

---

### 공식 3: 반전-공개 (Hook-Reveal)
트렌드 편승, 논쟁 유발 콘텐츠에 최적.

```
예상과 반대되는 주장/결과 제시
→ "왜?"라는 궁금증 유발
→ 의외의 이유 공개
→ "나도 해봐야지" CTA
```

**사용 시점**: 트렌드형, 챌린지형

## 바이럴 금지 패턴 (피해야 할 것들)

| 패턴 | 왜 안 되는가 |
|:---|:---|
| "안녕하세요 여러분~" 인사로 시작 | 첫 3초 낭비 → 이탈률 급등 |
| "오늘은 ___에 대해 알아볼게요" 예고 | 호기심 소진 → 바로 이탈 |
| "AI가 세상을 바꿀 것입니다" 서론 | 거창함 = 지루함 |
| CTA 2개 이상 | 분산 → 전환율 하락 |
| 조회수 낚시 후 정보 부실 | 저장률 0, 팔로우 전환 0 |
| 전문용어 나열 (풀어쓰기 없이) | 이탈률 급등 |
| 결론 없이 끝나는 콘텐츠 | 시청 완료율 하락 |

## 실전 바이럴 패턴 아카이브

> CDO가 실제 성과 데이터 기반으로 업데이트. 초기값은 업계 일반 패턴 기준.

### AI+인사이트 카테고리

**패턴 A: "몰랐던 AI 기능 공개"**
- 후킹: 비밀 공개 + 충격 통계
- 구조: 역순 대본법
- 메커니즘: 사회적 화폐 + 실용 가치
- 예상 시청 완료율: 높음 (실용 정보라 끝까지 봄)
- 적합 포맷: 30~45초 릴스

**패턴 B: "AI 착각 바로잡기"**
- 후킹: 반전 + 부정 어법
- 구조: 반전-공개 공식
- 메커니즘: 사회적 화폐 (내가 틀렸다는 걸 알게 됨)
- 예상 저장률: 높음 (나중에 확인하려고 저장)
- 적합 포맷: 인스타 릴스 + 캐러셀 후속

---

### AI+육아 카테고리

**패턴 C: "바쁜 부모를 위한 AI 꿀팁"**
- 후킹: 공감 고통 + 결과 먼저
- 구조: 문제-공감-해결 공식
- 메커니즘: 공감 고통 + 실용 가치
- 예상 공유율: 높음 (배우자·친구에게 공유)
- 적합 포맷: 30~45초 릴스

---

## 업데이트 규칙

- CDO가 실제 터진 콘텐츠 분석 후 이 파일에 패턴 추가
- 패턴은 "가설"과 "검증됨" 레이블로 구분
- 검증 기준: 평균 2배 이상 조회수 달성 콘텐츠 3개 이상에서 패턴 공통 관찰
