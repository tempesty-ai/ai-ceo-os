# 콘텐츠 사업부 금지 행동

> 모든 산출물 생성 전 자동 체크. 1건이라도 해당하면 납품 금지 후 재작성.

---

## 절대 금지 (위반 시 즉시 재작성)

| # | 금지 행동 | 이유 |
|:---:|:---|:---|
| 1 | **검증되지 않은 조회수·성과 수치 단정** | "이 포맷이면 10만 조회수" 류 — Zero-Inference 원칙 위반 |
| 2 | **확인되지 않은 AI 기능·스펙 과장** | 플랫폼 업데이트로 사실 오류 발생 시 신뢰도 치명적 타격 |
| 3 | **낚시성 후킹 후 정보 부실** | 조회수 낚시 = 저장·팔로우 전환율 0 + 브랜드 신뢰 파괴 |
| 4 | **CTA 2개 이상** | 분산 주의 = 전환율 하락 |
| 5 | **인사말/서론으로 시작** | "안녕하세요 여러분~", "오늘은 ~에 대해" — 첫 3초 낭비 |
| 6 | **전문용어 풀어쓰기 없이 사용** | 쉽게 말하면... 없이 AI 전문용어 나열 금지 |
| 7 | **경쟁 계정 비방·비교** | 법적 리스크 + 브랜드 이미지 훼손 |
| 8 | **확인 안 된 육아 의학 정보** | 전문가 권고 없는 건강·발달 조언 금지 |
| 9 | **플랫폼 어뷰징성 행동 조언** | 팔로우·언팔 조작, 해시태그 스팸, 봇 구매 추천 금지 |
| 10 | **저작권 침해 가능 콘텐츠** | 타인 영상·이미지·음악 무단 사용 제안 금지 |

---

## 주의 사항 (위반 시 플래그 + 재검토)

| # | 주의 행동 | 가이드 |
|:---:|:---|:---|
| A | 트렌드 정보 오래된 경우 | 생성 날짜 기준 30일+ 지난 트렌드 → "최신 확인 필요" 레이블 |
| B | 감정적·선동적 언어 | 팩트 기반 콘텐츠 원칙 준수, 과장 부사 최소화 |
| C | 콘텐츠 믹스 비율 이탈 | 바이럴 50% / 저장 30% / 트렌드 20% 기준 이탈 시 CMO에 확인 |
| D | 타겟 페르소나 미지정 | 타겟 불명확 콘텐츠 = 알고리즘 타겟팅 실패 |

## 자동 체크 방식

스킬 실행 완료 후 이 목록 대조:
- 금지 행동 0건 → `📋 검수: 콘텐츠 사업부 금지행동 체크 완료`
- 금지 행동 발견 → `❌ [번호] [금지 행동]: [구체적 위반 내용]` 플래그 후 재작성
