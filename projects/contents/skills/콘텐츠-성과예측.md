# 콘텐츠 성과 예측 스킬

> 콘텐츠 업로드 전 바이럴 가능성을 체계적으로 검증하는 파이프라인.
> CDO의 3가지 신호 교차 검증 방법론 적용.
> **Zero-Inference 원칙**: 데이터 없이 수치 예측 금지.

---

## 0단계: 자산 로드

### 필수 로드
- `projects/contents/context/viral-formula.md` (바이럴 공식·패턴 라이브러리)
- `projects/contents/context/platform-rules.md` (플랫폼별 알고리즘 신호)
- `projects/contents/rules/품질기준.md` (품질 기준선)
- `projects/contents/rules/금지행동.md` (금지 목록)

---

## 1단계: 콘텐츠 수신

사용자 입력: `$ARGUMENTS`

필요 정보:
- 콘텐츠 초안 (대본 / 기획서 / 개요)
- 타겟 플랫폼
- (선택) 비슷한 이전 콘텐츠 성과 데이터

**데이터가 없으면**: "이 예측은 데이터 기반이 아닌 패턴 가설입니다. [X] 데이터가 있으면 더 정확한 예측이 가능합니다" 레이블 필수 부착.

---

## 2단계: 3가지 신호 교차 검증

### 신호 1: 후킹 강도 (Hook Score)

체크 항목:
- [ ] 첫 3초에 후킹 배치 여부
- [ ] 사용된 후킹 타입 (12타입 중 어떤 것?)
- [ ] 후킹이 콘텐츠 본문과 일치하는가 (낚시성 여부)
- [ ] 후킹 개수 (3개 이상?)

**평가**: 상 / 중 / 하 + 근거 1줄

---

### 신호 2: 트렌드 일치도 (Trend Score)

체크 항목:
- [ ] 현재 플랫폼 트렌드 주제와 연관성
- [ ] 트렌드 편승 타이밍 (지금이 적기인가?)
- [ ] 경쟁 콘텐츠 포화도 (이미 너무 많은 비슷한 콘텐츠?)

**평가**: 상 / 중 / 하 + 근거 1줄

> 트렌드 정보가 없거나 30일 이상 지난 정보 → "최신 트렌드 확인 필요" 레이블

---

### 신호 3: 구조 완성도 (Structure Score)

체크 항목:
- [ ] 역순 설계 여부 (최종 감정 먼저 정의했는가?)
- [ ] 시청 완료율 높은 구조 부합 (`viral-formula.md` 공식 참조)
- [ ] CTA 1개인가?
- [ ] CEO 톤앤매너 준수?
- [ ] 전문용어 풀어쓰기 처리?

**평가**: 상 / 중 / 하 + 근거 1줄

---

## 3단계: 종합 판정

```
## 바이럴 가능성 예측

### 신호 점수
| 신호 | 점수 | 근거 |
|:---|:---:|:---|
| 후킹 강도 | 상/중/하 | [1줄] |
| 트렌드 일치도 | 상/중/하 | [1줄] |
| 구조 완성도 | 상/중/하 | [1줄] |

### 종합 판정
**바이럴 가능성**: [상 / 중 / 하]

**판정 근거**: [2~3줄]

**데이터 신뢰도**: [데이터 기반 / 패턴 가설 — 이유]
```

### 판정 기준
| 3개 신호 | 종합 판정 |
|:---|:---|
| 3개 모두 상 | 상 — 강하게 추천 |
| 2개 이상 상 | 중상 — 추천 |
| 1개만 상 | 중 — 개선 후 업로드 |
| 3개 모두 중 이하 | 하 — 재기획 권장 |

---

## 4단계: 개선 포인트

신호 중 "하" 또는 "중"이 있으면 구체적 개선 제안:

```
## 개선 포인트

1. [가장 중요한 개선 사항] — 예상 효과: [1줄]
2. [두 번째] — 예상 효과: [1줄]
3. [세 번째, 있으면] — 예상 효과: [1줄]
```

---

## 5단계: 최종 권고

```
## 최종 권고

[ ] 바로 업로드 추천
[ ] 개선 후 업로드 (위 개선 포인트 반영)
[ ] 재기획 권장 (구조적 문제)

플랫폼 최적 업로드 타이밍: [platform-rules.md 기준]
```
