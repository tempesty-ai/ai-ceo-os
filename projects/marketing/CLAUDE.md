# 마케팅 사업부 CLAUDE.md

> 이 사업부의 헌법. 본사 `CLAUDE.md`의 "고객 여정 6단계" 중 **1~3단계(첫 접점 → 신뢰 → 팔로우)**를 책임진다.
> 전환(4단계 이후)은 CXO/CPO와 공동 책임.

---

## 미션

잠재 팔로워를 콘텐츠로 끌어와 **팔로우·저장·DM 직전 단계까지** 연결한다.
"퍼널 상단을 두껍게, 중단을 새지 않게."

---

## 이 사업부가 책임지는 일

1. 인스타 릴스/숏폼 주간 기획 + 대본 (여정 1단계 후킹)
2. 캐러셀 신뢰 자산 제작 (여정 2단계)
3. 프로필 → 팔로우 전환 동선 설계 (여정 3단계)
4. 주간 채널 성과 리포트 (조회·저장·공유·프로필클릭·DM)
5. 벤치마크 계정 분석 + A/B 테스트 설계

---

## 담당 임원

- **주 담당**: CCO — 콘텐츠/후킹/톤 최종 검수
- **부 담당**: CMO — 퍼널 전환·도달·저장률 검증
- **크로스렌즈**: CDO(측정), CXO(1→2단계 감정 전환)

## 성공 지표 (KPI)

- **1단계 (유입)**: 릴스 주간 발행 수, 평균 도달·저장률
- **2단계 (신뢰)**: 프로필 방문 → 팔로우 전환율
- **3단계 (리드)**: DM 문의 수, 링크트리 클릭률

> 수익/결제 전환율은 CXO/CPO 영역. 이 사업부는 **팔로워 품질**까지.

## 크로스 사업부 협업 프로토콜

**참조**: `projects/통합-오케스트레이터.md`

### 언제 contents 사업부와 협업하는가
- 콘텐츠 생산 시: contents가 기획한 콘텐츠의 **팔로우 전환 관점 검수** 담당
- 주간 캘린더 시: contents 캘린더에 **고객 여정 단계 레이블** 추가
- 성과 분석 시: contents CDO 조회수 데이터 + marketing CDO 전환율 데이터 교차

### marketing이 contents에 요청하는 것
- "이 주제 조회수 관점에서 후킹 강화해줘" → contents CCO
- "이 콘텐츠 바이럴 가능성 예측해줘" → contents CDO

### contents가 marketing에 요청하는 것
- "이 대본이 팔로우 전환으로 이어지는가?" → marketing CCO
- "이 콘텐츠 고객 여정 몇 단계용인가?" → marketing CMO

### 단독 실행 조건
팔로우 전환·저장률·DM만 관련된 작업 → marketing 단독 실행 (통합 오케스트레이터 불필요)
