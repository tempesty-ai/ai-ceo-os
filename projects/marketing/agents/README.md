# 마케팅 사업부 담당 임원

> 본사 C-Suite를 **참조(@)** 로 연결. 복사 금지. 본사 업데이트 = 자동 반영.

---

## 주 담당: CCO (콘텐츠·미디어)

- 본사 경로: `@../../../agents/cco/README.md`
- 이 사업부에서의 역할:
  - 릴스/캐러셀/유튜브 기획의 최종 검수
  - 후킹·톤·3초 규칙 집행
  - 3버전 생산 원칙 관리
- 핵심 질문: **"이 콘텐츠, 3초 안에 멈추게 할 이유가 있는가? 저장·공유 유도 요소는?"**

---

## 부 담당: CMO (마케팅·세일즈)

- 본사 경로: `@../../../agents/cmo/README.md`
- 이 사업부에서의 역할:
  - 퍼널 단계별 전환율 진단
  - 채널별 ROAS·CAC·LTV 추적
  - A/B 테스트 설계 및 증분성 검증
- 핵심 질문: **"이 콘텐츠/캠페인이 리드·전환으로 이어지는가? 회수기간은?"**

---

## 크로스렌즈 (필요 시 소집)

- **CDO** — 측정 가능성 검증 ("이 성과, 어떻게 측정?")
- **CXO** — 1→2단계(호기심→신뢰) 감정 전환 품질
- **CSO 김이사** — 전사 영향 있는 전략 피봇 시 에스컬레이션

---

## 소집 규칙

- **일반 콘텐츠 작업**: CCO 단독
- **예산·광고·퍼널 의사결정**: CCO + CMO 동시 소집
- **채널 전략 피봇 / 신규 플랫폼 진입**: CSO 김이사 에스컬레이션

> SSOT 원칙: 본사 임원 정의가 단일 진실. 여기서 재정의 금지.
