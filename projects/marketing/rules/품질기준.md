# 마케팅 사업부 품질 기준선

> "적당히"는 없다. 아래 기준을 통과해야 완성.
> 모든 마케팅 산출물에 자동 적용된다.

---

## 릴스 대본

| 기준 | 합격 조건 |
|:---|:---|
| 첫 3초 후킹 | 질문/충격/반전 중 하나. filler(인사/감사) 절대 금지 |
| 분량 | 30~40초 (~120자 내외) |
| 모든 줄 | 인사이트 또는 꿀팁. 빈 문장 없음 |
| CTA | 정확히 1개 (저장/댓글/팔로우 중 택1) |
| CEO 톤 | `context/channel-profile.md` 보이스 가이드 일치 |
| 3버전 | 교육형·스토리텔링형·권위형 모두 존재 |
| 여정 단계 | 타겟 단계 명시 완료 |

## 캐러셀 원고

| 기준 | 합격 조건 |

## 성과 보고서

| 기준 | 합격 조건 |
|:---|:---|
| 수치 출처 | 모든 수치에 데이터 출처 명시 |
| 구조 | 요약 → 핵심 인사이트 3개 이내 → 액션 아이템 |
| 액션 아이템 | 담당자(CCO/CMO/CDO/CXO) 명시 |
| Zero-Inference | 추정 수치 없음. 데이터 없으면 "데이터 필요" |

## 전략/기획 문서

| 기준 | 합격 조건 |
|:---|:---|
