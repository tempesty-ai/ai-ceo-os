# 성과 분석 스킬

> 마케팅 사업부 **CDO 핵심 반복 작업**.
> 인스타그램 인사이트 데이터를 바탕으로 주간/월간 성과를 분석하고,
> 다음 주 콘텐츠 전략에 피드백을 준다.

---

## 언제 쓰나 (Trigger)

- "성과 분석해줘"
- "주간 리포트 만들어줘"
- "이번 주 어떤 콘텐츠가 잘됐어?"
- "저장률/도달률 정리해줘"
- "KPI 현황 어때?"
- "콘텐츠 성과 데이터 보고서"

---

## 전제 조건 (Zero-Inference)

이 스킬은 **실제 데이터 없이는 실행하지 않는다.**

데이터 없이 요청이 들어오면:
```
"성과 분석을 위해 인스타그램 인사이트 데이터가 필요합니다.
아래 형식으로 데이터를 붙여넣어 주세요:

- 기간:
- 콘텐츠별 도달 수:
- 콘텐츠별 저장 수:
- 콘텐츠별 공유 수:
- 프로필 방문 수:
- 팔로워 증감:
"
```

## 자동 로드할 것

- `projects/marketing/CLAUDE.md` (KPI 정의)
- `knowledge/channel-data/` (기존 채널 데이터, 있는 경우)

---

## 3-Phase 파이프라인

### Phase 1. 데이터 수집 확인

- 제공된 데이터 형식 확인
- 누락 지표 파악 → "N 지표가 없습니다. 없는 경우 해당 항목은 분석에서 제외합니다"
- 분석 기간 확인

### Phase 2. 분석 실행

**2-1. KPI 현황 정리**
- 전주 대비 변화율 (데이터가 있을 경우만)
- 상위/하위 콘텐츠 식별 (저장률 기준)

**2-2. 패턴 탐지**
- 저장률 높은 콘텐츠의 공통점
- 저장률 낮은 콘텐츠의 공통점
- 업로드 시간대/요일 패턴 (데이터 있는 경우)

**2-3. 이상 징후**
- 전주 대비 급락 (30% 이상 하락) 지표
- 콘텐츠 믹스 비율 이탈 확인 (AI+인사이트 70% / AI+육아 30%)

### Phase 3. 제안 도출

CDO는 데이터 해석까지. 전략 결정은 CMO.
- "데이터가 말하는 것: ..." (CDO 범위)
- "전략 결정 필요: ..." (CMO 에게 전달)
