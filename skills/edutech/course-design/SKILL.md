# 강의/코스 설계 스킬 (L.E.A.R.N 프레임워크)

> "배우면 좋겠다" 수준의 막연한 아이디어 → **판매 가능한 커리큘럼**으로 변환.
> 완강률·재구매율·상품 사다리까지 설계 완료본 산출.

---

## L.E.A.R.N 프레임워크

| 단계 | 의미 | 핵심 질문 |
|:---:|:---|:---|
| **L**earner | 학습자 프로파일 | "누가 배우는가? 지금 뭐가 답답한가?" |
| **E**nd Goal | 최종 목표 | "수강 후 무엇을 할 수 있는가?" |
| **A**rchitecture | 구조 설계 | "어떤 순서로 배워야 하는가?" |
| **R**etention | 완강률 부스터 | "끝까지 가게 하는 장치는?" |
| **N**ext Step | 수익화 구조 | "재구매/업셀 경로는?" |

## 2단계: E — 역방향 설계 (Backward Design)

```
[A] 최종 목표 정의
   "수강생이 이 강의를 끝내면 _______를 할 수 있다"

[B] 평가 기준 정의
   "그것을 어떻게 증명하는가? (과제/결과물/지표)"

[C] 학습 경험 설계
   "어떤 순서로 배워야 B가 가능해지는가?"
```

**금지**: 강사가 알려주고 싶은 것부터 나열 → 배열 → 커리큘럼. 이건 강사 중심. 결과 중심으로 역방향 설계할 것.

## 3단계: A — 커리큘럼 구조 출력

### 주차별 구조 템플릿

```markdown
## [강의명] 커리큘럼

### Week 1: [주차 주제]
**마일스톤**: 이 주가 끝나면 수강생이 [결과물]을 갖고 있다.

| # | 영상 제목 | 유형 | 길이 | 핵심 학습 |
|:---:|:---|:---:|:---:|:---|
| 1.1 | ... | 개념 | 8분 | ... |
| 1.2 | ... | 실습 | 20분 | ... |
| 1.3 | ... | 사례 | 12분 | ... |

**실습 과제**: [구체적 결과물]
**체크포인트**: [수강생이 다음 주로 넘어가기 전 확인할 것]

### Week 2: ...
```

### 영상 길이 가이드 (엄수)

| 유형 | 권장 길이 | 이유 |
|:---|:---:|:---|
| 개념 설명 | 7~10분 | 집중력 한계 |
| 실습 따라하기 | 15~25분 | 중간에 일시정지 반복 |
| 사례/스토리 | 10~15분 | 감정선 유지 |
| Q&A / 라이브 | 30~60분 | 깊이 있는 대화 |

### 원칙: 1영상 1핵심
- 한 영상은 **단 하나의 학습 목표**만 다룬다
- "이 영상 끝나면 수강생이 X를 할 수 있다" 한 줄로 요약 가능해야 함
- 여러 주제 섞으면 완강률 급락

### 원칙: 실습 비중 50% 이상
- 전체 영상의 최소 50%는 "직접 해보는" 실습형
- 개념만 듣는 강의 = 망치는 강의

## 4단계: R — 완강률 부스터 (최소 3개 적용)

| # | 부스터 | 효과 |
|:---|:---|:---|
| 1 | **주차별 잠금 해제** | FOMO + 주기적 재방문 |
| 2 | **마일스톤 배지/인증** | 성취감 + 공유 자산 |
| 3 | **과제 제출 + 피드백** | 인간적 연결 + 외부 압력 |
| 4 | **주간 라이브 Q&A** | 포기 방지 + 커뮤니티 |
| 5 | **수강생 커뮤니티** | 동기부여 + 사회적 증명 |
| 6 | **진도율 리마인드** | 기술적 푸시 |
| 7 | **완강 인증서** | 외부 증명 + 다음 상품 유인 |

**선택 기준**: 학습자 프로파일(Q1~Q5) 기반으로 가장 맞는 3개+ 선택.

---

## 5단계: N — 수익화 구조 (상품 사다리)

```markdown
## 상품 사다리

### Tier 0: 무료 리드마그넷
- 형태: [PDF/체크리스트/미니강의]
- 목적: 이메일/연락처 확보
- 예상 전환율: 10~30%

### Tier 1: 입문 상품 (5~15만원)
- 이름:
- 구성: (짧은 강의 or 미니 부트캠프)
- 업셀 타이밍: 수강 N주차 완료 후

### Tier 2: 메인 상품 (30~70만원) ⭐ 주력
- 이름:
- 구성: (풀 커리큘럼 + 커뮤니티 + 피드백)
- 업셀 타이밍: 수강 완료 N일 후 or 성과 달성 직후

### Tier 3: 프리미엄 (100만원+)
- 이름:
- 구성: (1:1 코칭 / 그룹 멘토링 / VIP 케어)
- 업셀 조건: 메인 상품 완강 + NPS 9~10
```

---

## 6단계: Cross-Lens 자가 점검 + Inversion

```markdown
## Cross-Lens 자가 점검
- CMO: 이 커리큘럼이 세일즈 페이지에서 팔 만한가?
- CFO: 제작/운영 비용 대비 수익 구조가 성립하는가?
- CXO: 수강생 입장에서 온보딩~첫 성공까지 며칠 걸리는가?

## 이 커리큘럼이 실패하는 3가지 시나리오
1. 완강률 20% 이하 → 조기 경고: Week 1 이탈률 체크
2. 과제 제출률 30% 이하 → 조기 경고: 과제 난이도 재조정
3. 재구매율 0% → 조기 경고: Tier 2 수강자 NPS 추적
```

## 금지 사항

- **강사 중심 나열**: "내가 알려주고 싶은 것" 순서로 배치
- **이론 과잉**: 실습 없이 개념만 30분 이상
- **과장 약속**: "이 강의 하나로 월 1000만원!" 같은 허위 기대치
- **가격 숨기기**: 상담 강제 유도 목적의 가격 비공개
- **완강률 무시**: 판매만 설계하고 끝까지 가는 장치 없음

## 검수 기준

| ID | 기준 | 합격 조건 | 가중치 |
|:---|:---|:---|:---:|
| E1 | 학습자 프로파일 Q1~Q5 | 5가지 모두 구체적으로 답변 | 필수 |
| E2 | 역방향 설계 3단계 | 최종 목표·평가 기준·학습 경험 명시 | 필수 |
| E3 | 1영상 1핵심 | 각 영상이 단일 학습 목표 | 필수 |
| E4 | 실습 비중 50%+ | 전체 영상 중 실습형 비율 | 필수 |
| E5 | 완강률 부스터 3개+ | 7가지 중 최소 3개 적용 | 필수 |
| E6 | 수익화 구조 | 상품 사다리(무료→입문→메인→프리미엄) 포함 | 필수 |
| E7 | 영상 길이 가이드 | 개념 7~10분, 실습 15~25분 준수 | 권장 |
| E8 | Before→After 명확 | 각 모듈의 수강 전/후 변환이 1줄 요약 가능 | 권장 |
| E9 | Cross-Lens + Inversion | 자가 점검 + 실패 시나리오 3가지 | 필수 |

**합격 기준**: 필수 항목 전체 통과 + 권장 항목 50% 이상
