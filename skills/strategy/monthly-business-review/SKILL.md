# 월간 사업 리뷰 스킬

> 매월 말/초 1회, 전사 OKR·P&L·AI 비용·전략 조정을 **한 번에** 점검.
> 김이사(CSO)가 주도하되, 필요 시 CMO/CFO/CDO 서브에이전트 병렬 소집(Tier 2).

---

## 언제 쓰는가

- 매월 1~3일 (전월 마감 직후)
- 분기 말 종합 리뷰 전 베이스라인 확보
- 주요 전략 변경 직전 현황 스냅샷

**축약 금지**: "간단히" 요청 시에도 아래 섹션 전부 포함. 대신 길이는 압축.

---

## 0단계: 페르소나 활성화

`.claude/agents/cso.md` 서브에이전트로 전환. 필요 시 Tier 2 소집 판단:

- 광고 이상치 있음 → CMO + CDO 병렬 소집
- 수익성 급변 → CFO + CDO 병렬 소집
- 완강률/NPS 이상 → CPO + CXO 병렬 소집

---

## 1단계: 자산 로드 (Phase 1 — 병렬)

### 필수
- `CLAUDE.md`
- `context/mission-vision.md` (전략 테제)
- `tasks/todo.md`, `tasks/lessons.md`, `tasks/predictions.md`, `tasks/session_handoff.md`

### 조건부
- `knowledge/channel-data/` (채널 성과)
- `execution/` 스크립트 결과물 (월별 집계)
- 본인 OKR 파일 (있으면 `tasks/okr.md` 등)

---

## 2단계: OKR 점검

```markdown
## OKR 점검 ([YYYY년 MM월])

### Objective 1: [목표]
| Key Result | 목표 | 현재 | 달성률 | 상태 |
|:---|:---:|:---:|:---:|:---:|
| KR 1.1 | ... | ... | N% | 🟢/🟡/🔴 |
| KR 1.2 | ... | ... | N% | 🟢/🟡/🔴 |

**김이사 판단**: (목표 대비 진행, 차질 원인, 다음 달 조정)

### Objective 2: ...
```

**판단 기준**:
- 달성률 70%+ = 🟢
- 40~70% = 🟡 (원인 분석 필수)
- <40% = 🔴 (목표 재정의 or 리소스 재배분 검토)

## 3단계: P&L 리뷰

```markdown
## P&L ([YYYY년 MM월])

### 매출
| 상품/채널 | 매출 | 비중 | 전월 대비 |
|:---|:---:|:---:|:---:|
| [상품 A] | (출처) | N% | +N% |
| [상품 B] | (출처) | N% | -N% |
| **합계** | | 100% | |

### 비용
| 항목 | 금액 | 비중 | 전월 대비 | 비고 |
|:---|:---:|:---:|:---:|:---|
| 광고비 | | | | |
| 인건비 | | | | |
| 인프라/SaaS | | | | |
| 수수료/세금 | | | | |
| 기타 | | | | |
| **합계** | | 100% | | |

### 실수익
- 매출 - 총비용 = **실수익**: (출처)
- 매출 대비 실수익률: N%
- 전월 대비: +/-N%

### 핵심 질문
- 이번 달 돈을 번 1순위 경로는?
- 이번 달 돈을 가장 많이 쓴 1순위 항목은?
- 그 항목의 ROI가 성립하는가?
```

**Zero-Inference**: 모든 숫자에 출처(스크립트/대시보드/수동 입력) 명시.

## 4단계: AI 운영 비용 추적

> 1인 CEO가 AI 시스템을 돌리는 비용이 매출의 몇 %인지 매월 점검.

```markdown
## AI 운영 비용 ([YYYY년 MM월])

| 항목 | 이번 달 | 전월 | 증감 | 비고 |
|:---|:---:|:---:|:---:|:---|
| Claude API | $XX | $XX | +N% | |
| 인프라 (EC2/Vercel/Render) | $XX | $XX | | |
| 기타 API (YouTube/Meta/Google/Notion 등) | $XX | $XX | | |
| SaaS 구독 (Notion/Figma/자동화 도구) | $XX | $XX | | |
| **합계** | **$XX** | **$XX** | | |

### ROI 판단
- AI 운영비 / 매출 = N%
- 5% 초과 시 에스컬레이션, 원인 분석 필수
- 전월 대비 50%+ 증가 시 원인 분석 필수
```

## 7단계: Cross-Lens 자가 점검 + Inversion (필수)

```markdown
## Cross-Lens 자가 점검
- CMO: (마케팅 관점 영향 1줄)
- CFO: (재무 관점 영향 1줄)
- CPO: (상품 관점 영향 1줄)

## 이 다음 달 계획을 망치는 3가지 시나리오
1. ... → 조기 경고:
2. ... → 조기 경고:
3. ... → 조기 경고:
```

## 검수 기준

| ID | 기준 | 합격 조건 | 가중치 |
|:---|:---|:---|:---:|
| E1 | OKR 점검 포함 | 분기/월간 OKR 달성률 수치 명시 | 필수 |
| E2 | P&L 섹션 포함 | 매출·비용·실수익이 테이블로 정리 | 필수 |
| E3 | 전략 조정안 포함 | 유지/중단/시도 3분류 + 제안 최소 3건 | 필수 |
| E4 | AI 운영 비용 추적 | Claude API·인프라·기타 전월 대비 | 필수 |
| E5 | Zero-Inference | 모든 수치 출처 명시 | 필수 |
| E6 | 액션 아이템 담당/기한 | 모든 액션에 담당+기한 명시 | 필수 |
| E7 | 전월 대비 변화율 | 주요 지표에 증감률(%) 표기 | 권장 |
| E8 | 리스크 플래그 | 위험 기준 초과 항목에 경고 | 권장 |
| E9 | Cross-Lens + Inversion | 자가 점검 + 3가지 시나리오 포함 | 필수 |

**합격 기준**: 필수 항목 전체 통과 + 권장 항목 50% 이상
