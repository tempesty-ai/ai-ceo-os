# 임원별 Lessons 폴더

> 각 임원이 자기 도메인의 실수/교훈을 누적하는 **장기 기억 저장소**입니다.
> `agents/_shared/cognitive-core.md` § 6.1 규칙에 따라, 모든 임원은 보고 시작 전 자기 lessons 파일을 자동 로드합니다.
> 전사 공통 교훈은 상위 폴더의 `tasks/lessons.md`에 기록.

## 파일 구조

- `cmo.md` — 마케팅+세일즈
- `cfo.md` — 재무+경영전략
- `cto.md` — 기술+인프라
- `cdo.md` — 데이터+분석
- `coo.md` — 운영+조직
- `cco.md` — 콘텐츠+미디어
- `cpo.md` — 제품+교육
- `cxo.md` — 고객경험+커뮤니티

## 기록 형식

```markdown
## YYYY-MM-DD — [한 줄 요약]
- 상황:
- 잘못된 판단:
- 근본 원인 (5 Whys):
- 교훈:
- 향후 적용 규칙:
```

## 활용

- **인간 임원**이 "전에 그거 망했잖아"를 기억하는 것과 동일한 메커니즘
- 6개월 누적 후 → 임원별 반복 실수 패턴 분석 → cognitive-core 또는 페르소나 강화
