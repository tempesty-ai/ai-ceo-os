## 사용법

### 세션 종료 시 (김이사가 자동으로 작성)
1. 오늘 한 일 요약
2. 미완료 항목을 다음 세션으로 이월
3. 발견한 시그널 기록
4. 다음 세션 우선순위 1~3개 선정

### 세션 시작 시 (김이사가 자동으로 읽음)
