# 마케팅 성과 분석 파이프라인

> 광고/마케팅 데이터를 받아 ROAS·CPA·전환율을 한 번에 분석.

---

## 목적
"이번 주 광고 어땠어?" 한 줄에서 시작해, ROAS/CPA/전환율 분석 + 액션 아이템 도출.

## 입력
- 분석 기간 (지난 7일/30일/이번 달)
- (선택) 비교 기간

## Phase 1 — 데이터 수집
- `execution/ads_performance_check.py` 호출
- `knowledge/channel-data/` 참조

## Phase 2 — 분석
김이사 + CMO + CDO 협업:
1. **퍼널 단계별 전환율** (노출→클릭→가입→결제)
2. **채널별 ROAS** (인스타/유튜브/구글/페북)
3. **CPA 추세** (지난 기간 대비)
4. **이상치 감지** (평소 대비 변동 큰 지표)

## Phase 3 — 검증 + 액션
| 에이전트 | 체크 |
|:---|:---|
| CMO | ROAS 해석, 증액/감액 판단 |
| CFO | 수수료 차감 후 실수익 |
| CDO | 통계적 유의성 |

## 출력
- 요약 (1장)
- 핵심 인사이트 3개 이내
- 액션 아이템 (담당자, 기한 포함)

## 검수 기준
- [ ] 모든 숫자 데이터 출처 명시
- [ ] 액션 아이템에 담당자/기한
- [ ] 추측 0건
- [ ] 이상치 자동 표시
