# 월간 사업 리뷰 파이프라인

> 매월 1일 실행. OKR 점검 + P&L 리뷰 + 다음 달 전략 조정.

---

## 목적
한 달에 한 번, 전사 성과를 점검하고 다음 달 우선순위를 잡는다.

## 입력
- 지난 달 데이터 (매출, 광고비, 콘텐츠 성과, 고객 수)
- `CEO/목표.md` (OKR)

## Phase 1 — 데이터 수집
- `execution/`의 모든 데이터 스크립트 실행
- `outputs/{지난달}/`의 산출물 전부 스캔
- `tasks/lessons.md`의 지난 달 교훈

## Phase 2 — 4-Part 리뷰

### Part 1: OKR 진척도
- Objective별 KR 달성률
- 미달 시 원인

### Part 2: P&L
- 매출 vs 비용 vs 이익
- 채널별 ROI

### Part 3: 콘텐츠/마케팅 성과
- TOP 3 콘텐츠
- WORST 3 콘텐츠
- 다음 달 시도할 것

### Part 4: 사람/팀
- 이번 달 새로 만든 스킬
- 이번 달 발견한 시그널

## Phase 3 — 검증 (Tier 3)
8명 임원 전원 소집:
- 각 임원이 본인 영역에서 R1 분석
- R2 반박 라운드
- 김이사 최종 결론

## 출력
- 월간 리뷰 보고서 (4-Part)
- 다음 달 OKR 조정안
- 우선순위 TOP 5
- 예측 (다음 달 매출/성장)

## 검수 기준
- [ ] OKR 모든 KR 달성률 표기
- [ ] P&L 수치 데이터 출처 명시
- [ ] 다음 달 액션 5개 이내
- [ ] 예측에 확신도(%) 표기
- [ ] `tasks/predictions.md`에 예측 자동 기록
