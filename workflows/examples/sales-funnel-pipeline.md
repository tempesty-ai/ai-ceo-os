# 세일즈 퍼널 분석 파이프라인

> 유입~결제 전환율을 단계별로 점검하고 병목을 찾는 파이프라인.

---

## 목적
세일즈 퍼널의 각 단계 전환율을 점검하고, 가장 큰 병목을 찾아 해결책 제안.

## 입력
- 분석 기간
- 퍼널 단계 (CLAUDE.md 고객 여정 6단계 자동 참조)

## Phase 1 — 데이터 수집
- 인스타 인사이트 / 카페 통계 / 결제 데이터 / 카톡 대화 수
- `knowledge/channel-data/` 참조

## Phase 2 — 단계별 전환율 계산
김이사 + CMO + CXO 협업:

| 단계 | 전환율 |
|:---|---:|
| 1. 호기심 (콘텐츠 노출 → 프로필 방문) | {{N%}} |
| 2. 신뢰 (프로필 → 카페/오픈채팅 가입) | {{N%}} |
| 3. 자신감 (가입 → 무료 체험) | {{N%}} |
| 4. 통제감 (체험 → 핵심 상품 결제) | {{N%}} |
| 5. 효능감 (결제 → 만족/재구매) | {{N%}} |
| 6. 해방감 (재구매 → 추천/팬덤) | {{N%}} |

## Phase 3 — 병목 식별 + 해결책
| 에이전트 | 체크 |
|:---|:---|
| CMO | 가장 낮은 전환 단계 식별 |
| CXO | 그 단계의 마찰 지점 분석 |
| CCO | 콘텐츠로 해결 가능한 부분 |

## 출력
- 단계별 전환율 표
- 병목 단계 1개 + 원인 3개
- 해결책 3개 + 우선순위

## 검수 기준
- [ ] 모든 단계 전환율 데이터 출처 명시
- [ ] 병목 단계 명확
- [ ] 해결책에 우선순위
- [ ] 추측 0건
