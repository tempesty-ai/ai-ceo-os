# 유튜브 기획 파이프라인

> 주제 → 유튜브 영상 기획서(롱폼+쇼츠) 자동 생성

---

## 목적
유튜브 영상 1편당 기획서(제목/썸네일 카피/구조/대본 개요)를 한 번에 생성.

## 입력
- 주제
- 영상 길이 (롱폼 8~15분 / 쇼츠 60초 이내)

## Phase 1 — 리서치
| 에이전트 | 역할 |
|:---|:---|
| CCO | 유사 주제 인기 영상 패턴 분석 |
| 데이터 참조 | `knowledge/channel-data/` |
| 트렌드 리서치 | (선택) 웹 검색 |

## Phase 2 — 기획서 작성
김이사 통합:
1. **제목 후보 5개** (CTR 최적화)
2. **썸네일 카피 3개** (텍스트 + 표정 + 색)
3. **인트로 후킹** (15초 내)
4. **본문 구조** (3~5 chapter)
5. **클로징 + CTA**
6. **대본 개요** (chapter별 핵심 메시지)

## Phase 3 — 검증
| 에이전트 | 체크 |
|:---|:---|
| CCO | 후킹 강도, 브랜드 톤 |
| CMO | 전환 동선, CTA 적정성 |
| CDO | 예상 CTR/시청 지속률 |

## 출력
- 기획서 1장
- 제목 후보 5개
- 썸네일 카피 3개
- 대본 개요
- 촬영 체크리스트

## 검수 기준
- [ ] 제목 후보 최소 5개
- [ ] 인트로 15초 내 후킹
- [ ] 본문 chapter 3~5개
- [ ] CTA 명확
- [ ] 예상 시청 지속률 50%+
